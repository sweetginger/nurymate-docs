## 1. 전략 개요

> **우선순위 재정의 (2026-03-25 전략 회의 확정)**: 신규 창업자 타겟 확보는 중장기 과제로 유지하되, **단기 최우선 타겟은 셀메이트 기존 고객사 중 온오프라인 몰을 운영하는 디자이너 브랜드**로 확정되었다. 본 문서는 신규 창업자 전략을 담되, 아래 0절에 디자이너 브랜드 버티컬 전략을 선행 배치한다.

셀메이트CM의 중장기 주력 타겟 중 하나는 **쇼핑몰 신규 창업자**다. 기존 `early-customer-acquisition.md`가 카페24 불만 유저 전환 및 셀메이트 고객사 중심이라면, 본 문서는 처음부터 셀메이트CM를 선택하도록 설계하는 신규 창업자 전략을 다룬다.

**포지셔닝 메시지**: "시장에 있는 서비스 중 가장 유연하고 손품이 적게 들며, 판매에 필요한 모든 기능을 갖춘 솔루션"

## 0. 최우선 타겟: 셀메이트 디자이너 브랜드 버티컬 (2026-03-25 확정)

> 신규 창업자보다 **기존 셀메이트 고객사**를 먼저 공략한다. 버티컬을 좁게 장악한 후 확장하는 원칙에 따른다.

### 전략 근거

| 항목 | 내용 |
|------|------|
| **타겟** | 셀메이트 고객사 중 온라인 자사몰 + 오프라인 매장 동시 운영 디자이너 브랜드 |
| **시장 규모** | 한국 패션 브랜드 시장 2025년 **50.9조 원** (성장 중) |
| **진입 논거** | 셀메이트 기존 접점 활용 → 아웃리치 비용 최소화 |
| **니즈** | 온오프라인 연계(재고·정산·POS) + 자사몰 매출 확대 |
| **공략 순서** | 매장 수 많은 브랜드 먼저 → 레퍼런스 확보 → 동일 버티컬 내 확산 |

### 왜 신규 창업자보다 기존 셀메이트 고객사인가

```
신규 창업자 타겟                셀메이트 디자이너 브랜드 타겟
─────────────                  ──────────────────────────
신규 인지도 형성 필요            셀메이트 접점으로 즉시 접근 가능
전환 동기 불명확                 온오프라인 연계 고통 명확 (전환 동기 강함)
낮은 초기 GMV                   기존 운영 브랜드 → 즉시 GMV 발생
레퍼런스 부재                   셀메이트 레퍼런스로 신뢰 기반 출발

→ 초기 BEP 도달 속도 느림       → 초기 BEP 도달 속도 빠름
```

### 실행 방안

1. **셀메이트 내부 협력**: 영업·CS 팀과 공동으로 기존 고객사 대상 셀메이트CM 전환 프로그램 기획
2. **타겟 리스트 작성**: 셀메이트 고객사 중 온오프라인 몰 보유 브랜드를 매장 수 기준으로 정렬 → 상위 50개사 우선 공략
3. **맞춤 제안서**: 온오프라인 통합 운영 절감 효과(시간·비용)를 수치로 제시
4. **레퍼런스 확보**: 전환 1호 브랜드 성공 사례를 즉시 케이스 스터디로 제작 → 동일 버티컬 내 바이럴

### 확장 논리

```
셀메이트 디자이너 브랜드 버티컬 장악
         │
         ▼
온오프라인 연계 레퍼런스 다수 확보
         │
         ▼
패션 D2C 카테고리 내 셀메이트CM = "온오프 통합 1등 플랫폼" 포지셔닝
         │
         ▼
카페24 이탈 판매자 → 신규 창업자 순으로 확장
```

## 2. 한국 신규 쇼핑몰 창업 시장 규모

| 지표 | 현황 |
|------|------|
| 연간 통신판매업 신규 신고 건수 | 약 50만 건 (2024년 공정위 통계 추정) |
| 이커머스 솔루션 선택 신규 창업자 추정 | 연 10~20만 개 (소규모 D2C 포함) |
| 주요 플랫폼 선택 분포 (추정) | 카페24 30%, 아임웹 25%, 스마트스토어 20%, 기타 25% |
| 첫 1년 내 폐업률 | 약 50% (창업 초기 매출 실패가 주요 원인) |

**셀메이트CM 기회**: 연 10~20만 신규 창업자 중 상위 5~10%인 "성공 가능성 있는 창업자" 10,000~20,000명을 1차 타겟으로 설정.

## 3. 신규 창업자 페르소나

### 페르소나 A: 패션 인스타 창업자
- **배경**: 인스타그램 팔로워 3,000~10,000명, 패션 콘텐츠 운영 중
- **현 상황**: DM으로 주문받거나 스마트스토어 운영 중
- **목표**: 자사몰 전환, 브랜드 이미지 구축
- **플랫폼 선택 기준**: 예쁜 디자인, 간단한 셋업, 인스타 연동
- **셀메이트CM 매칭 포인트**: AI 30분 셋업, 인스타그램 SNS 연동, 무료 시작

### 페르소나 B: 디지털 노마드형 창업자
- **배경**: 직장 퇴사 후 부업/창업, 해외 판매 목표
- **현 상황**: 한국·일본 동시 판매 희망, 기술 숙련도 중간
- **목표**: 한국 + 일본 시장 동시 공략
- **플랫폼 선택 기준**: 글로벌 결제, 현지화, 운영 자동화
- **셀메이트CM 매칭 포인트**: 일본 결제 네이티브, 단위 현지화, AI 번역

### 페르소나 C: 직장인 부업 창업자
- **배경**: 현직 직장인, 주말 부업으로 핸드메이드/빈티지 판매
- **현 상황**: 아임웹·스마트스토어 고려 중, 비용 민감
- **목표**: 최소 비용으로 시작, 매출 나면 키우기
- **플랫폼 선택 기준**: 무료 또는 저렴한 시작, 간단한 관리
- **셀메이트CM 매칭 포인트**: 0% 수수료 구간, 단순 UX, AI 온보딩

## 5. 셀메이트CM의 신규 창업자 차별 가치 제안

```
"비교 없이 바로 시작하세요"

1. 30분 AI 셋업 — 개발자·디자이너 없이 오늘 오픈
2. 월 GMV 500만 원까지 0% 수수료 — 매출 날 때까지 무료
3. 일본·글로벌 확장 준비 완료 — 성장하면 해외도 바로
```

## 6. 유입 채널 전략

### 6.1 네이버 검색 기반

- **타겟 키워드**: "쇼핑몰 창업 방법", "자사몰 만들기", "온라인 쇼핑몰 플랫폼 비교"
- **콘텐츠 전략**: 네이버 블로그 + 포스트에 "셀메이트CM로 30분 만에 쇼핑몰 만들기" 시리즈
- **네이버 카페**: 온라인 창업 카페(쇼핑몰 창업, 부업 성공기 등) 활동 + 사례 공유

### 6.2 유튜브 창업 콘텐츠

- **타겟 채널**: 쇼핑몰 창업 유튜버, 부업 콘텐츠 크리에이터, 패션 D2C 성공 사례
- **협력 방식**: 협찬 리뷰 + 창업 가이드 콘텐츠 공동 제작
- **자체 채널**: "셀메이트CM로 쇼핑몰 창업하기" 튜토리얼 시리즈

### 6.3 창업 지원 기관 제휴

| 제휴 기관 | 접근 방법 |
|---------|---------|
| 소상공인시장진흥공단 | 창업 지원 프로그램 공식 파트너 등록 |
| 창업진흥원 | 예비 창업패키지 연계 쇼핑몰 구축 지원 |
| 지역 창업센터 | 강의·워크샵 제공 (셀메이트CM 셋업 실습) |
| 여성기업종합지원센터 | 여성 창업자 패션·뷰티 쇼핑몰 창업 지원 |

## 7. 온보딩 → 첫 매출 → 락인 퍼널 설계

```
[인지]
유튜브 "30분 셋업" 영상 / 네이버 블로그 리뷰 / 창업 카페 추천
    │
    ▼
[관심]
셀메이트CM 랜딩페이지 방문 → "무료로 시작하기" CTA
    │
    ▼
[가입]
이메일 가입 → AI 온보딩 (30분 목표)
    │
    ▼
[활성화]
스토어 개설 완료 → 웰컴 쿠폰 자동 발급 → 지인 공유 → 첫 주문
    │
    ▼
[정착]
첫 주문 경험 → 추가 기능 활성화 → 플랜 업그레이드
    │
    ▼
[락인]
GMV 증가 → 데이터 축적 → 전환 비용 상승 → 장기 고객
```
