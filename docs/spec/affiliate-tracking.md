## 1. 개요

누리메이트 판매자가 인플루언서·제휴 파트너에게 개인화된 추적 링크를 발급하고, 해당 링크를 통한 유입에서 결제 완료까지의 전환 퍼널을 8개 핵심 지표로 측정하는 기능.

### 1.1 사용 시나리오

- 인플루언서에게 협찬 상품 제공 → 고유 링크 발급 → 팔로워 클릭 추적
- 어필리에이트 파트너에게 커미션 지급 기반 마련
- A/B 테스트: 채널별·콘텐츠별 전환율 비교

---

## 2. 추적 링크 구조

### 2.1 링크 포맷

```
https://{스토어도메인}/p/{상품슬러그}?ref={파트너ID}&utm_source={채널}&utm_medium=affiliate
```

**예시:**
```
https://mybrand.nurymate.com/p/blue-sneakers?ref=influencer_001&utm_source=instagram
```

### 2.2 파트너 ID 체계

| 항목 | 규칙 |
|------|------|
| 포맷 | `{타입}_{고유번호}` (예: `inf_001`, `aff_042`) |
| 길이 | 최대 32자 |
| 유효기간 | 기본 90일 (갱신 가능) |
| 쿠키 유지 | 30일 (Last-click Attribution) |

---

## 3. 8개 핵심 추적 지표

| # | 지표명 | 정의 | 측정 방법 |
|---|--------|------|---------|
| 1 | **방문수 (Sessions)** | 링크를 통한 총 세션 수 (동일 사용자 재방문 포함) | 세션 쿠키 기반 카운팅 |
| 2 | **방문자수 (Unique Visitors)** | 링크를 통한 고유 방문자 수 (중복 제거) | 디바이스 핑거프린팅 + 쿠키 조합 |
| 3 | **상품조회수 (Product Views)** | 링크 방문 후 특정 상품 상세 페이지 조회 수 | 상품 페이지 뷰 이벤트 |
| 4 | **체류시간 (Time on Site)** | 링크 방문자의 평균 사이트 체류 시간 | 세션 시작·종료 타임스탬프 |
| 5 | **카트 추가 (Add to Cart)** | 링크 방문 후 장바구니에 상품 추가한 횟수 | `add_to_cart` 이벤트 |
| 6 | **좋아요 (Product Wishlist)** | 링크 방문 후 찜/위시리스트 추가 수 | `add_to_wishlist` 이벤트 |
| 7 | **결제 시도 (Checkout Initiated)** | 링크 방문 후 결제 프로세스 시작 횟수 | `begin_checkout` 이벤트 |
| 8 | **결제 완료 (Purchase Completed)** | 링크 방문 후 실제 결제 완료 건수 및 금액 | `purchase` 이벤트 + 주문 DB |

### 3.1 전환 퍼널 시각화

```
방문수 (100%)
    │ → 상품조회율
방문자 중 상품 조회 (~70%)
    │ → 카트 추가율
카트 추가 (~30%)
    │ → 좋아요율 (별도 경로)
위시리스트 추가 (~20%)
    │
결제 시도 (~15%)
    │ → 결제 완료율
결제 완료 (~10%)
```

## 5. 기술 구현

### 5.1 이벤트 수집 구조

```
[셀러 스토어 프론트엔드]
    │ 커스텀 이벤트 발생 (add_to_cart, begin_checkout 등)
    ▼
[누리메이트 Analytics SDK]
    │ ref 파라미터 포함 이벤트 전송
    ▼
[Analytics 수집 서버]
    │
    ├─ ref 파라미터 → 파트너 ID 매핑
    ├─ 쿠키에 파트너 ID 저장 (30일)
    └─ 이벤트 DB 저장
```

### 5.2 Attribution 방식

| 방식 | 설명 | 채택 여부 |
|------|------|---------|
| Last-click | 마지막 클릭 링크에 전환 기여 | ✅ 기본 채택 |
| First-click | 최초 클릭 링크에 기여 | 🔲 Phase 2 |
| Linear | 모든 터치포인트에 균등 배분 | 🔲 Phase 2 |

### 5.3 쿠키 정책

- **쿠키 키**: `nm_ref` (nurymate referral)
- **유효기간**: 30일
- **SameSite**: Lax (크로스사이트 요청에서도 GET 요청은 전송)
- **GDPR/개인정보보호**: 쿠키 동의 팝업과 연계
