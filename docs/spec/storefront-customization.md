## 1. 개요 및 배경

### 1.1 문제 정의

한국 D2C 브랜드, 특히 패션·라이프스타일 디자이너 브랜드들은 **카페24를 기반으로 온라인 스토어를 운영**하고 있다. 그러나 카페24의 구조적 한계로 인해 브랜드 고유의 아이덴티티를 스토어에 충분히 담지 못하고 있다.

**핵심 불만 지점:**
- 템플릿 기반 구조 → 구조 변경 불가, 섹션 위치 고정
- 스마트 디자인 에디터 자유도 낮음 → 이미지 비율·여백·타이포그래피 세밀 조정 어려움
- 기획전 페이지 스타일링 옵션 빈약 → 브랜드 특색 없는 획일화된 레이아웃
- 코드 수정은 가능하나 배포 후 카페24 업데이트로 깨지는 케이스 반복
- 모바일/PC 별도 설정이 분리되지 않거나 동기화 충돌

**결과:** 동일한 템플릿을 쓰는 수백 개의 스토어가 레이아웃·폰트·색상 구성이 유사해 브랜드 차별화가 어려움.

### 1.2 누리메이트의 포지션

누리메이트는 **헤드리스 커머스 아키텍처**를 기반으로 스토어프론트와 백엔드를 완전히 분리한다. 이로 인해:

- 프론트엔드(스토어프론트)는 어떤 구조로든 자유롭게 구성 가능
- 셀러는 코드 지식 없이도 비주얼 에디터로 깊은 수준의 커스터마이징 수행 가능
- 개발자 셀러는 HTML/CSS/JS를 직접 수정해 완전한 자유도 확보

**타겟 셀러:**
- 카페24에서 이탈을 고려하는 디자이너 브랜드 (한국 시장 1순위)
- 브랜드 아이덴티티를 스토어에 투영하고 싶은 D2C 셀러
- 기획전·컬렉션 단위로 다른 무드의 페이지를 운영하고 싶은 셀러

## 2. 커스터마이징 가능 영역

### 2.1 상품 리스트 페이지 (PLP — Product List Page)

#### 2.1.1 그리드 옵션

| 옵션 | 설명 | 비고 |
|------|------|------|
| 2열 그리드 | 이미지 크게 강조, 럭셔리·고가 브랜드 적합 | 기본값 (모바일) |
| 3열 그리드 | 균형 잡힌 배치, 범용 | 기본값 (PC) |
| 4열 그리드 | 상품 수 많은 편집샵·다품종 브랜드 적합 | |
| Masonry 그리드 | 이미지 세로 비율 혼합, 인스타그램 감성 | |
| 리스트형 (1열) | 상품명·설명 텍스트 비중 높을 때 | |

- 모바일/PC 각각 독립 설정 가능
- 사용자(구매자)가 직접 그리드를 전환할 수 있는 토글 UI 제공 여부 셀러가 결정

#### 2.1.2 카드 레이아웃 커스터마이징

**이미지 영역:**
- 이미지 비율 설정: 1:1 (정사각), 3:4 (세로), 4:3 (가로), 자유 비율
- 호버 시 추가 이미지 전환 (hover swap) 활성화 여부
- 이미지 fit 옵션: `cover` / `contain` / `fill`
- 이미지 테두리 반경 설정 (0 ~ 50px, 혹은 원형)

**텍스트 영역:**
- 상품명 표시 위치: 이미지 하단 / 이미지 위 오버레이 / 이미지 내 하단 고정
- 텍스트 정렬: 왼쪽 / 가운데 / 오른쪽
- 줄 수 제한 (1줄 말줄임 / 2줄 / 제한 없음)
- 서브 텍스트 항목 선택: 브랜드명, 색상/옵션명, 간략 설명 중 0~2개 선택

**가격 표시:**
- 정가/할인가 표시 순서 설정
- 할인율(%) 뱃지 표시 여부 및 스타일 (원형/사각/없음)
- 품절 시 표시: "품절" 텍스트 / 이미지 딤처리 / 숨김 중 선택
- 가격 폰트 강조 여부 (굵게, 색상 별도 지정)

**카드 전체:**
- 카드 간 여백(gap) 조정
- 카드 하단 패딩 조정
- 배경색 설정 (개별 카드 vs 전체 리스트 배경)

#### 2.1.3 필터 UI

| 항목 | 옵션 |
|------|------|
| 필터 위치 | 사이드바 (왼쪽) / 상단 수평 바 / 상단 드롭다운 |
| 필터 항목 | 카테고리, 색상, 사이즈, 가격대, 브랜드, 소재, 커스텀 태그 — 개별 활성화/비활성화 |
| 색상 필터 UI | 컬러 스와치 / 텍스트 체크박스 선택 |
| 모바일 필터 | 바텀 시트 / 전면 오버레이 / 상단 칩 중 선택 |
| 필터 적용 방식 | 즉시 적용 / "필터 적용" 버튼 클릭 후 적용 |

- 커스텀 필터 항목 추가: 셀러가 직접 정의한 속성(예: "소장각", "한정판" 등)을 필터로 등록 가능

#### 2.1.4 정렬 옵션

- 기본 정렬 설정 (신상품 / 인기순 / 가격 낮은순 / 가격 높은순 / 리뷰 많은순)
- 정렬 항목 표시/숨김 개별 설정
- 정렬 UI 위치: 필터 바 우측 / 독립 드롭다운

### 2.2 기획전(이벤트) 페이지

기획전은 브랜드의 **시즌 무드·컨셉**을 가장 강하게 표현하는 지면이다. 카페24에서는 기획전 페이지가 거의 HTML 직접 편집에 의존하거나 템플릿이 획일화되어 있는 반면, 누리메이트는 빌더 방식으로 구성한다.

#### 2.2.1 배너 레이아웃

| 레이아웃 | 설명 |
|---------|------|
| 풀스크린 배너 | 뷰포트 100% 히어로 이미지, 몰입형 |
| 반반 (50/50) | 왼쪽 이미지 + 오른쪽 텍스트/상품, 또는 반대 |
| 그리드 배너 | 2~4개 이미지 타일, 각각 링크 설정 가능 |
| 텍스트 전용 | 이미지 없이 컬러 배경 + 타이포그래피 중심 |
| 비디오 배너 | mp4/YouTube 임베드, 자동재생·음소거 설정 |

- 배너별 텍스트 오버레이 설정: 제목, 부제, CTA 버튼
- 배너 전환 애니메이션 선택: 페이드 / 슬라이드 / 줌 / 없음

#### 2.2.2 상품 배열 규칙

- 수동 배열: 드래그앤드롭으로 상품 순서 직접 지정
- 자동 배열: 조건 기반 (신상품순 / 판매량순 / 재고 있는 상품 우선 등)
- 하이라이트 상품: 특정 상품을 카드 크기 2배로 강조 표시
- 상품 수 제한: "이 기획전에서 N개만 표시" 설정

#### 2.2.3 테마 색상 자동 적용

- 기획전별 **테마 팔레트** 설정 (배경색, 텍스트색, 버튼색 3개)
- 설정한 테마 팔레트가 해당 기획전 페이지 전체에 자동 적용
- 브랜드 프리셋에서 저장된 팔레트 불러오기 가능
- 다크/라이트 모드 별도 설정 지원

#### 2.2.4 카운트다운 타이머

- 기획전 종료 시각 입력 → 타이머 자동 표시
- 타이머 UI 스타일: 숫자형 / 아날로그형 / 텍스트형
- 타이머 위치: 배너 상단 / 배너 하단 / 상품 리스트 상단
- 종료 후 처리: 페이지 숨김 / "종료된 기획전" 메시지 / 자동 리다이렉트

### 2.3 홈 히어로섹션

홈 화면의 첫 인상은 브랜드 아이덴티티를 전달하는 가장 중요한 지면이다.

#### 2.3.1 히어로 타입 선택

| 타입 | 설명 |
|------|------|
| 단일 이미지 | 정적 이미지 1장, 심플하고 고급스러운 느낌 |
| 슬라이더 | 이미지 2~10장 자동/수동 전환 |
| 비디오 배경 | mp4 업로드 또는 YouTube/Vimeo URL, 자동재생·루프 |
| 컬러 배경 + 타이포 | 이미지 없이 배경색 + 텍스트로 구성 |
| Split (반반) | 왼쪽 이미지 + 오른쪽 텍스트 |

#### 2.3.2 슬라이더 세부 설정 (슬라이더 타입 선택 시)

- 자동 재생 속도: 2초 ~ 10초 사이 조정
- 전환 효과: 슬라이드 / 페이드 / 줌아웃
- 탐색 컨트롤 표시: 화살표 / 점(dot) 인디케이터 / 둘 다 / 없음
- 각 슬라이드별 개별 링크 설정

#### 2.3.3 오버레이 텍스트

- 위치: 9분할 그리드 포지셔닝 (좌상/중상/우상/좌중/중앙/우중/좌하/중하/우하)
- 텍스트 구성: 헤드라인 / 서브헤드 / CTA 버튼 (각각 독립 표시/숨김 설정)
- 색상: 텍스트색, 텍스트 배경 딤(dim) 투명도 조정
- 폰트 크기: 상대값(px) 또는 Responsive 단위 (vw)

#### 2.3.4 CTA 버튼

- 텍스트 직접 입력
- 링크 대상: 카테고리 / 기획전 / 외부 URL / 앵커 스크롤
- 버튼 스타일: 채움(filled) / 외곽선(outlined) / 텍스트 링크
- 버튼 색상, 테두리 반경, 패딩 조정

#### 2.3.5 모바일 별도 설정

- 모바일 전용 이미지/비디오 업로드 (PC와 독립)
- 텍스트 위치 모바일 독립 설정
- 히어로 높이: PC와 모바일 각각 vh/px 단위로 지정
- 모바일에서 비디오 자동재생 여부 (배터리·데이터 고려)

### 2.4 상품 상세 페이지 (PDP — Product Detail Page)

#### 2.4.1 이미지 갤러리

- 썸네일 위치: 왼쪽 세로형 / 하단 가로형 / 없음
- 메인 이미지 전환: 클릭 / 호버 / 스와이프(모바일)
- 줌 기능: 호버 줌 / 클릭 확대 모달 / 비활성화
- 이미지 개수 제한 없음 (카루셀 자동 처리)

#### 2.4.2 상품 정보 레이아웃

- 이미지/정보 배치: 좌이미지-우정보 (기본) / 우이미지-좌정보 / 상이미지-하정보 (모바일 기본)
- 섹션 순서 드래그앤드롭 재배치: 가격, 옵션, 배송정보, 리뷰, 상세설명 등
- 스티키 구매 버튼: 스크롤 시 하단 고정 여부

#### 2.4.3 상세설명 에디터

- 리치 텍스트 에디터 (Bold, 이미지 삽입, 링크, 표 등)
- HTML 직접 편집 모드 전환
- 상품 상세 섹션 분리: 탭 구조 (상품정보 / 배송·반품 / 리뷰) 또는 세로 스크롤 구조 선택

---

### 2.5 장바구니 / 결제 페이지

#### 2.5.1 장바구니 UI

- 사이드 드로어(슬라이드아웃) vs 전용 페이지 이동 중 선택
- 장바구니 아이템 카드 스타일: 이미지 크기, 텍스트 정보 표시 항목 설정
- 쿠폰/포인트 입력창 표시 여부 및 위치

#### 2.5.2 결제 페이지

- 1페이지 결제(Single Page Checkout) vs 단계별 결제 선택
- 게스트 결제 허용 여부
- 배송지/결제정보 폼 필드 표시 항목 조정 (필수/선택 항목 설정)
- 브랜드 컬러 적용 범위: 버튼색, 체크박스, 프로그레스 바

> 참고: 결제 페이지는 PG사 연동 정책 범위 내에서 커스터마이징 가능. 카드 입력 폼 영역은 PG사 제공 iframe으로 스타일 자유도 제한.

## 3. 편집 방식

누리메이트는 셀러 기술 수준에 따라 세 가지 편집 방식을 지원한다.

### 3.1 비주얼 에디터 (No-Code)

**대상**: 비개발자, 디자이너, 일반 브랜드 운영자

**구성:**
- 캔버스 좌측: 실시간 스토어 미리보기 (PC/모바일 전환 가능)
- 캔버스 우측: 섹션별 속성 패널 (설정값 슬라이더·드롭다운·컬러피커)
- 섹션 추가: 사이드바에서 섹션 블록 드래그앤드롭으로 삽입
- 섹션 순서: 위아래 화살표 또는 드래그로 재배치
- 실시간 반영: 설정 변경 즉시 미리보기에 반영 (저장 전 상태)

**지원 섹션 블록 예시:**
- 히어로 배너 / 슬라이더
- 상품 그리드 / 캐러셀
- 텍스트 + 이미지 조합
- 기획전 배너
- 공지/배너 스트립
- 푸터

### 3.2 코드 직접 편집 (Pro Mode)

**대상**: 개발자 셀러, 에이전시, 고급 사용자

**제공 범위:**
- HTML/Liquid-like 템플릿 문법 직접 편집
- CSS 오버라이드 (스코프드 CSS 지원, 플랫폼 기본 스타일 덮어쓰기 가능)
- JavaScript 커스텀 스크립트 삽입 (안전 정책 내)
- 코드 에디터 내장 (Monaco Editor 기반, 신택스 하이라이팅)
- Git 연동 (향후 로드맵): 외부 레포 연결 → 배포 자동화

### 3.3 병행 모드 (Hybrid)

- 비주얼 에디터로 기본 구성 → 특정 섹션만 코드 오버라이드
- 오버라이드 적용된 섹션은 비주얼 에디터에서 "코드 편집 중" 뱃지 표시
- 코드 오버라이드 초기화 버튼으로 비주얼 설정값 복원 가능
- 비주얼 편집 변경 사항이 코드 오버라이드 영역을 덮어쓰지 않도록 격리

## 5. 카페24 vs 누리메이트 자유도 비교

### 5.1 구조적 차이

카페24는 **모놀리식 커머스(Monolithic Commerce)** 구조다. 프론트엔드 템플릿이 플랫폼 내부에 종속되어 있어, 셀러는 플랫폼이 허용하는 범위 내에서만 수정할 수 있다.

누리메이트는 **헤드리스 커머스(Headless Commerce)** 구조다. 프론트엔드(스토어프론트)와 백엔드(주문·재고·결제)가 API로만 연결되어 있어, 프론트엔드는 완전히 독립적으로 구성할 수 있다.

### 5.2 기능별 비교표

| 커스터마이징 항목 | 카페24 | 누리메이트 |
|----------------|-------|-----------|
| **레이아웃 구조 변경** | 템플릿 선택만 가능, 섹션 추가 불가 | 비주얼 에디터로 섹션 자유 추가·재배치 |
| **상품 그리드 열 수** | 2열/3열 제한적 선택 | 2/3/4열·Masonry·리스트형 |
| **이미지 비율** | 템플릿 고정값 | 비율 자유 설정 |
| **히어로 섹션 타입** | 슬라이더 중심, 타입 변경 어려움 | 슬라이더/단일/비디오/Split 자유 선택 |
| **기획전 페이지 테마** | HTML 편집 필요 | 팔레트 선택만으로 전체 적용 |
| **카운트다운 타이머** | 별도 앱 설치 필요 | 기본 제공 |
| **폰트 설정** | 제공 폰트 목록 제한적, 웹폰트 업로드 불가 | 구글 폰트 + 웹폰트 업로드 |
| **색상 팔레트** | 템플릿별 제한, 전역 변수 개념 없음 | 토큰 기반 전역 팔레트 시스템 |
| **모바일 독립 설정** | PC 설정이 모바일에 종속 | PC/모바일 완전 독립 설정 |
| **코드 직접 편집** | 가능하나 플랫폼 업데이트 시 덮어써짐 | 코드 오버라이드 + 비주얼 격리 보장 |
| **레이아웃 프리셋 저장** | 미지원 | 스냅샷 저장·전환 |
| **결제 페이지 스타일** | PG사 기본 UI 고정 | 브랜드 컬러 범위 내 커스터마이징 |
| **Git 연동 배포** | 미지원 | 로드맵 (v2 이후) |

### 5.3 셀러 수준별 도달 가능한 커스터마이징 깊이

```
카페24
├── 비개발자: 템플릿 선택 + 텍스트/이미지 교체만 가능
├── 반개발자: 스마트 디자인 에디터 제한적 활용
└── 개발자: HTML/CSS 수정 (플랫폼 업데이트에 취약)

누리메이트
├── 비개발자: 비주얼 에디터 → 섹션 구성, 그리드, 색상, 폰트 전부 가능
├── 반개발자: 비주얼 + 코드 오버라이드 혼합 → 세밀한 디테일 조정
└── 개발자: 풀 코드 편집 → 플랫폼과 독립적으로 완전한 커스텀 스토어프론트
```
