## 1. 개요 및 설계 원칙

### 1.1 목적

누리메이트 셀러가 자사몰을 개설하고 첫 매출을 달성하기까지의 전체 UX 흐름을 설계한다. 셀러 전환율(가입 → 첫 매출)을 극대화하고, 구매자에게도 신뢰감 있는 쇼핑 경험을 제공하는 것이 핵심 목표다.

### 1.2 핵심 설계 원칙

| 원칙 | 설명 |
|------|------|
| **Time-to-Value 단축** | 회원가입 후 30분 이내 스토어 오픈 가능하도록 설계 |
| **Progressive Disclosure** | 필수 설정만 먼저, 고급 설정은 나중에 |
| **AI-First 온보딩** | AI 어시스턴트가 셋업 전 과정에서 가이드 제공 |
| **Mobile-First** | 한국 모바일 커머스 65%, 일본 60% 비율 감안 |
| **Local Context** | 한국/일본 각 시장에 맞는 결제·배송 기본값 자동 설정 |

### 1.3 타겟 셀러 페르소나

- **김지은 (32세, 패션 D2C)**: 인스타그램 1만 팔로워 보유, 카페24 이탈 검토 중. 일본 진출 희망. 기술 지식 낮음.
- **박준혁 (38세, 식품 스타트업)**: 스마트스토어 운영 중, 독립몰 필요성 느낌. 결제 수수료 절감 목표.
- **Tanaka Hiroshi (41세, 일본 잡화)**: 한국 D2C 플랫폼 탐색 중. 일본어 UI·엔화 결제 필수.

## 2. 자사몰 생성 전 단계

### 2.1 전체 사전 단계 플로우

```
[랜딩페이지] → [회원가입] → [이메일 인증] → [비즈니스 프로필] → [플랜 선택] → [도메인 설정] → [Setup Wizard 진입]
```

### 2.2 회원가입 플로우

#### Step 1: 가입 방식 선택

```
+-----------------------------------------+
|  누리메이트에 오신 것을 환영합니다        |
|  자사몰을 시작해 보세요                   |
|                                         |
|  [ Google로 시작하기 ]                   |
|  [ Kakao로 시작하기  ] (KR 기본 노출)   |
|  [ LINE으로 시작하기 ] (JP 기본 노출)   |
|  ─────────── 또는 ──────────────        |
|  이메일: ___________________________     |
|  비밀번호: _________________________     |
|  [ 무료로 시작하기 ]                     |
|                                         |
|  이미 계정이 있으신가요? [로그인]         |
+-----------------------------------------+
```

- **국가 자동 감지**: IP 기반으로 한국/일본 소셜 로그인 버튼 우선 노출
- **소셜 로그인 후**: 추가 정보 입력 단계(비즈니스 이름, 연락처)로 이동
- **이메일 가입 후**: 인증 메일 발송, 인증 완료 전까지 스토어 생성 불가

#### Step 2: 비즈니스 기본 정보

```
+-----------------------------------------+
|  비즈니스 정보를 알려주세요              |
|  (나중에 언제든 수정할 수 있어요)        |
|                                         |
|  스토어 이름 *                           |
|  [____________________________]         |
|  → URL 미리보기: yourstore.nurymate.com  |
|                                         |
|  주요 판매 카테고리 *                    |
|  ○ 패션/의류  ○ 뷰티  ○ 식품            |
|  ○ 홈/리빙    ○ 전자기기  ○ 기타        |
|                                         |
|  주요 판매 국가 *                        |
|  ☑ 한국      ☑ 일본      □ 기타         |
|                                         |
|  현재 판매 경험                          |
|  ○ 처음 시작하는 분    ○ 타 플랫폼 이전  |
|  ○ 이미 운영 중        ○ 에이전시/대행사  |
|                                         |
|  [다음 단계로 →]                         |
+-----------------------------------------+
```

- 스토어 이름 입력 시 실시간 URL 미리보기 및 중복 체크
- 판매 국가 선택에 따라 이후 단계에서 통화·배송·결제 기본값 자동 설정

#### Step 3: 이메일 인증 (소셜 로그인 제외)

- 인증 메일 발송 후 6자리 코드 입력 or 링크 클릭
- 5분 타임아웃, 재발송 버튼 제공
- 인증 완료 시 컨페티(Confetti) 애니메이션 + "인증 완료! 이제 스토어를 만들어 보세요" 문구

### 2.3 플랜 선택

#### 플랜 비교 화면

```
+------------------------------------------------------------------+
|  어떤 플랜으로 시작하시겠어요?                                    |
|  언제든지 변경할 수 있어요. 첫 30일은 무료입니다.                  |
|                                                                  |
|  [STARTER]         [GROWTH]              [PRO]                  |
|  무료               월 ₩29,000            월 ₩79,000             |
|  ─────────          ─────────             ─────────              |
|  GMV 5백만↓ 0%수수료 GMV 5백만 초과 1%    GMV 5백만 초과 0.5%    |
|  기본 테마 5종      모든 테마              모든 테마 + 커스텀       |
|  상품 100개         상품 무제한            상품 무제한              |
|  기본 결제 수단     모든 결제 수단         모든 결제 수단           |
|  -                 SNS 연동               SNS 연동 + 광고 분석    |
|  -                 일본어 자동 번역        다국어 완전 지원         |
|  -                 -                     전담 CS 지원             |
|                                                                  |
|  [Starter로 시작]  [Growth로 시작 ★추천]  [Pro로 시작]           |
|                                                                  |
|  * 30일 무료 체험 후 선택한 플랜으로 자동 전환                     |
|  * 언제든 플랜 변경 및 해지 가능                                   |
+------------------------------------------------------------------+
```

- Growth 플랜 기본 하이라이트 (추천 배지)
- 연간 결제 시 2개월 할인 옵션 토글
- 플랜 선택 후 결제 정보 입력 (카드번호, 간편결제)
  - 30일 무료 기간 명확히 고지: "30일 동안 무료, 이후 월 ₩29,000 청구"

### 2.4 도메인 설정

#### 도메인 옵션 선택 화면

```
+------------------------------------------+
|  스토어 주소를 설정하세요                  |
|                                          |
|  ○ 누리메이트 무료 도메인 사용             |
|    yourstore.nurymate.com                |
|    → 즉시 사용 가능, 무료                  |
|                                          |
|  ○ 내 도메인 연결 (추천)                   |
|    www.myfashionstore.com                |
|    → 이미 갖고 있는 도메인 연결            |
|    [도메인 입력] + [연결 방법 안내]         |
|                                          |
|  ○ 새 도메인 구매                          |
|    .com / .co.kr / .shop 등              |
|    [도메인 검색]                           |

|                                          |

+------------------------------------------+
```

- 커스텀 도메인 연결: DNS 설정 가이드 자동 제공 (호스팅사별 스크린샷 포함)
- 도메인 연결 검증 자동 polling (최대 48시간, 상태 뱃지 표시)
- 일본 셀러: `.jp` 도메인 옵션 우선 표시

## 3. 셋업 단계

### 3.1 Setup Wizard 개요

총 5단계로 구성, 각 단계 완료 시 상단 프로그레스 바 업데이트.

```
[기본정보] → [테마 선택] → [상품 등록] → [결제 설정] → [오픈 준비]
  ● ────────── ○ ────────── ○ ────────── ○ ────────── ○
```

- 각 단계는 건너뛰기 가능 (상단 "나중에 하기" 버튼)
- 우측 상단 AI 어시스턴트 버블: 단계별 맥락 안내 제공
- 이탈 시 진행상태 자동 저장 ("이어서 하기" 재진입 가능)

### 3.2 Step 1: 스토어 기본 정보

```
+-----------------------------------------------------+
|  스토어 기본 정보를 입력해 주세요           1/5      |
|  ●────○────○────○────○                              |
|                                                     |
|  스토어 이름 *           로고 업로드                 |
|  [My Fashion Store   ]  [📁 파일 선택]              |
|                         권장: 400x400px, PNG         |
|                                                     |
|  스토어 설명                                         |
|  [트렌디한 여성 패션을 선보이는          ]           |
|  [My Fashion Store입니다.              ]            |
|  (0/500자)                                          |
|                                                     |
|  연락처 이메일 *         고객센터 전화번호            |
|  [hello@store.com    ]  [010-0000-0000 ]            |
|                                                     |
|  스토어 위치 (발송지)                                |
|  [서울특별시 강남구 ▾ ] [상세주소          ]         |
|                                                     |
|  사업자 정보 (선택)                                  |
|  [사업자 등록번호        ]  [상호명          ]        |

```

- 로고 업로드 시 즉시 미리보기 (스토어 헤더에 어떻게 보이는지)
- AI 어시스턴트: "스토어 설명에 키워드를 포함하면 검색 노출에 유리해요" 팁 제공

- 일본 판매 대상 상품: 엔화 가격 자동 환산 필드 추가 표시

  ```
  셀러: 네
  AI: [사이즈 가이드 템플릿 자동 생성 → 편집 후 저장]
  ```

### 4.4 첫 매출 달성 축하 플로우

1. 첫 주문 발생 시 어드민 내 모달 팝업 + 컨페티 효과
2. 알림 이메일 발송: "축하합니다! 첫 주문이 들어왔어요"
3. SNS 공유 유도: "첫 매출 달성 인증샷" 이미지 자동 생성 (공유 가능)
4. 다음 목표 제시: "이제 10건 달성을 향해 나아가 보세요!"

## 6. 모바일 UX 고려사항

### 6.1 한국·일본 모바일 커머스 현황

| 지표 | 한국 | 일본 |
|------|------|------|

| 주요 SNS | 인스타그램, 카카오 | 인스타그램, LINE |
| 평균 구매 결정 시간 | 짧음 (임펄스 구매) | 다소 긴 편 (신중) |

### 6.2 모바일 셀러 어드민

- **바텀 네비게이션**: 홈 / 주문 / 상품 / 통계 / 설정
- **주문 알림**: 푸시 알림 즉시 수신, 탭 한 번으로 배송 처리
- **상품 등록 모바일 최적화**: 카메라 바로 촬영 → AI 배경 제거 → 즉시 등록

### 6.3 모바일 구매자 경험

- **섬네일 로딩**: WebP 포맷 + lazy loading으로 빠른 이미지 로딩
- **결제 최적화**: 카카오페이/토스/PayPay 원탭 결제 (팝업 최소화)
- **터치 타겟**: 최소 44x44px (Apple HIG 기준)
- **바텀 시트 UI**: 옵션 선택, 장바구니 담기 등을 바텀 시트로 처리

### 6.4 PWA 지원

- Service Worker 기반 오프라인 탐색 (상품 캐시)
- 홈 화면 추가 지원 (스플래시 스크린, 아이콘)
- 푸시 알림 (결제 완료, 배송 업데이트)
