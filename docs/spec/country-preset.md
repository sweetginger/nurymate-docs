## 1. 개요 및 문제 정의

### 1.1 배경

이커머스 판매자가 새로운 국가로 판매를 확장할 때 다음과 같은 설정 영역에서 오류·누락이 빈번하게 발생한다.

| 설정 영역 | 현재 문제 |
|-----------|-----------|
| 결제수단 | 해당 국가 구매자가 선호하는 결제 수단 미지원 → 구매 이탈 |
| 세금 설정 | 부가세/소비세 요율 미설정 → 세금 미징수 또는 이중 부과 |
| 환불 정책 | 국가별 법정 청약철회 기간 미준수 → 법적 분쟁 위험 |
| 법적 고지문 | 표시의무 항목 누락 → 행정처분·과태료 위험 |
| 배송 설정 | 해당 국가 배송사·배송비 미설정 → 주문 접수 후 배송 불가 |

### 1.2 문제 정의

판매자가 새로운 국가로 판매를 시작할 때 **국가별 법적 요건과 비즈니스 관행을 개별적으로 파악하여 설정**해야 한다. 이 과정에서:

- **누락**: 법적 필수 고지문을 모르거나 빠뜨림
- **오류**: 세금 요율 잘못 입력, 환불 기간 법정 기준 미충족
- **지연**: 설정 방법 파악에 시간 소요 → 판매 시작 지연
- **비용**: 법률 전문가 자문 비용 발생

### 1.3 솔루션 개요

**"판매 국가 선택 → 프리셋 자동 적용"** 구조를 통해 판매자가 법적·비즈니스적 요건을 일일이 조사하지 않고도 해당 국가에서 즉시 판매 가능한 상태를 만든다.

**벤치마크 — Shopify Markets:**
Shopify는 국가별 시장(Market) 개념을 도입하여 언어·통화·도메인·세금·가격 설정을 국가 단위로 분리 관리하고, 동적 세금 포함 가격 표시(Dynamic tax-inclusive pricing)를 지원한다. 셀메이트CM은 여기에 더해 **법적 고지문 자동 생성** 및 **로컬 결제수단 네이티브 통합**을 차별점으로 추가한다.

「전자상거래 등에서의 소비자보호에 관한 법률」 제17조 기준:

```
[기본 환불 정책 — 한국]
- 청약철회 기간: 상품 수령일로부터 7일 이내 (이유 불문)
- 단순 변심 반품: 왕복 배송비 구매자 부담

- 청약철회 불가 사유:
  - 사용 또는 일부 소비로 가치 감소된 경우
  - 포장 훼손으로 상품 가치 현저히 감소된 경우
  - 시간 경과로 재판매 어려운 경우
  - 복제 가능한 디지털 상품 포장 훼손 시
```

#### 2.2.3 환불 정책 (특定商取引に関する法律 — 통신판매 조항)

일본 특정상거래법상 통신판매는 청약철회(クーリング・オフ) 의무 적용 대상이 아니나, 반품 특약을 명시해야 한다.

```
[기본 환불 정책 — 일본]
- 반품 기간: 상품 수령일로부터 8일 이내 (미개봉·미사용)
- 반품 배송비: 상품 불량·오배송의 경우 판매자 부담
- 단순 변심 반품: 구매자 부담 (별도 고지)
- 반품 특약 없을 경우: 법정 8일 이내 반품 허용 의무 발생
- 교환: 재고 있는 경우에 한해 대응
- 환불 처리: 반품 확인 후 7영업일 이내
```

#### 2.2.4 세금 설정

| 항목 | 설정값 |
|------|--------|
| 소비세 (消費税) | 10% (일반) / 8% (경감세율 — 식품·음료·신문 등) |
| 가격 표시 방식 | 税込価格 (세금 포함 표시) 의무화 (2021년~ ) |
| 인보이스 제도 | 적격청구서 (インボイス) 발행 지원 (2023년 10월 도입) |
| 영수증 (領収書) | PDF 발행 기능 제공 (구매자 요청 시) |
| 경감세율 적용 품목 | 판매자 직접 지정 (식품 카테고리 선택 시 자동 제안) |

> 인보이스 제도: 2023년 10월부터 시행된 적격청구서 등 보존 방식. 면세 사업자는 인보이스 발행 불가. 셀메이트CM은 판매자의 등록 사업자 여부에 따라 인보이스 발행 가능 여부를 자동 표시한다.

#### 2.2.5 영수증 (領収書) 발행 기능

일본에서는 구매 후 영수증 발행 요청이 빈번하며, 법인 경비 처리 목적으로 특정 형식을 요구하는 경우가 많다.

```
[領収書 발행 항목]
- 수신: 구매자 성명 또는 법인명
- 금액: 税込合計金額
- 但し書き (용도 기재란): "○○代として" 형식
- 발행일: 결제일 기준
- 발행자: 판매자 상호 및 주소
- 발행 방식: PDF 다운로드 (주문 완료 페이지 + 마이페이지)
```

#### 2.2.6 필수 법적 고지문 — 일본

특定商取引法 제11조 (通信販売) 필수 표시사항:

```
[特定商取引法に基づく表記 — 필수 항목]
1. 販売業者 (판매사업자명)
2. 運営統括責任者 (운영 총괄 책임자 성명)
3. 所在地 (주소)
4. 電話番号 (전화번호)
5. メールアドレス
6. 販売URL
7. 販売価格 (상품별 페이지에 표시)
8. 販売価格以外に必要な費用 (배송비·手数料 등)
9. 代金の支払時期・方法
10. 商品の引渡し時期
11. 返品・キャンセルに関する特約
12. 動作環境 (디지털 상품의 경우)
```

> 이 페이지(特定商取引法に基づく表記)는 일본에서 판매하는 모든 온라인 쇼핑몰에 법적으로 필수이며, 쇼핑몰 상단 또는 하단에서 접근 가능해야 한다. 셀메이트CM은 판매자 정보 입력을 기반으로 이 페이지를 자동 생성하고 일본어로 표시한다.

## 3. 자동 적용 vs 판매자 확인 필요 항목 분류

### 분류 기준

| 구분 | 정의 | 예시 |
|------|------|------|
| 자동 적용 (Auto) | 법적 표준값이 존재하거나 합리적 기본값 설정 가능 | 세율, 법적 고지문 구조, 청약철회 기간 |
| 판매자 확인 필요 (Review) | 판매자 개별 정보·선택이 반드시 필요한 항목 | 상호, 주소, 배송비, 결제수단 계약 여부 |
| 선택 활성화 (Optional) | 기본값 비활성화이나 판매자 선택으로 켤 수 있는 항목 | 특정 결제수단, 경감세율 품목 지정 |

- 전체 항목 수·자동 적용 수·확인 필요 수를 상단에 요약 표시

### 4.3 재적용 및 초기화

- 프리셋 적용 후 개별 설정을 수정한 경우, "프리셋 기본값으로 초기화" 버튼 제공
- 프리셋 재적용 시 현재 커스터마이즈 내용이 덮어쓰여짐을 경고 표시

## 5. 커스터마이즈 가능 범위

### 5.1 커스터마이즈 원칙

프리셋은 **합리적 기본값**을 제공하되, 판매자의 비즈니스 상황에 따라 유연하게 조정 가능해야 한다. 단, **법적 최소 요건 이하로의 설정은 차단**한다.

### 5.2 커스터마이즈 가능 항목

| 항목 | 커스터마이즈 가능 범위 | 제한 사항 |
|------|----------------------|----------|
| 배송비 | 금액·무료 기준·지역별 추가비 자유 설정 | 없음 |
| 배송 기간 | 일수 조정 가능 | 없음 |
| 반품 기간 | 8일 이상으로만 설정 가능 (일본 기준) | 법정 최소 8일 미만 불가 |
| 청약철회 기간 | 7일 이상으로만 설정 가능 (한국 기준) | 법정 최소 7일 미만 불가 |
| 세금 (VAT/소비세) | 경감세율 품목 지정 가능 | 기본 세율(10%) 임의 변경 불가 |
| 결제수단 | 계약된 결제수단 on/off | 계약 미완료 수단은 활성화 불가 |
| 법적 고지문 | 추가 문구 입력 가능 | 필수 항목 삭제 불가 |
| 영수증 양식 | 但し書き(용도) 기본값 설정 가능 | 필수 기재 항목 삭제 불가 |

### 5.3 법적 요건 보호 로직

- 법정 최소 요건 이하 설정 시도 시 인터랙티브 경고 표시
- 경고 메시지 예시: "일본 특정상거래법상 반품 특약 미기재 시 법정 8일 이내 반품을 반드시 허용해야 합니다. 최소 8일 이상으로 설정해주세요."
- 경고 무시 후 저장 차단 (법적 보호 항목은 저장 자체를 막음)

## 6. 지원 국가 확장 로드맵

### Phase 1 — 한국 (MVP, 2026 Q2)

- 대상: 한국 내 판매 (기본값)
- 우선순위: 모든 판매자의 기본 출발점
- 포함 항목: 한국 프리셋 전체 (§2.1)
- 목표: 통신판매업 신고 정보 연동, 부가세 신고 자료 자동 생성

### Phase 2 — 일본 (2026 Q3~Q4)

- 대상: 한국 판매자의 일본 크로스보더 판매
- 포함 항목: 일본 프리셋 전체 (§2.2)
- 추가 필요 작업:
  - 일본어 법적 고지문 번역 검수 (법무 전문가)
  - 일본 결제 대행사 계약 연동 (콘비니·PayPay 등)
  - 인보이스 제도 연동 (适格請求書발행사업자 등록번호 입력)

### Phase 3 — 동남아시아 (2027 Q1~Q2)

| 국가 | 우선 지원 이유 | 주요 결제수단 |
|------|--------------|-------------|
| 태국 | 한류 소비 강세, K-뷰티 수요 | PromptPay, 편의점 결제 |
| 싱가포르 | 영어권, 규제 투명 | PayNow, 신용카드 |
| 말레이시아 | 이슬람 할랄 인증 필요성 | DuitNow, FPX |
| 인도네시아 | 동남아 최대 시장 | QRIS, GoPay, OVO |

- 주요 추가 요건: 현지 소비자보호법, 데이터 현지화 규정, 환율 정책

### Phase 4 — 글로벌 (2027 Q3~)

- 미국: CCPA(캘리포니아 소비자 프라이버시법), 주별 세금 Nexus 설정
- EU: GDPR 고지문, VAT OSS(One Stop Shop) 신고
- 호주: GST 10%, Australian Consumer Law 14일 반품
- 영국: UK GDPR, VAT 20%

### 7.3 업데이트 분류 및 대응 속도

| 분류 | 정의 | 대응 기한 |

| 일반 (Standard) | 유예기간 있는 법령 개정 | 유예기간 내 배포 |

### 7.4 판매자 책임 범위 명시

- 셀메이트CM은 법적 고지문의 기술적 구조와 기본 템플릿을 제공하나, **판매자가 입력한 개별 정보(상호·주소·특약 조건 등)의 정확성은 판매자 책임**임을 이용약관에 명시
- 법률 변경 후 업데이트 알림 발송 시점부터 미적용에 따른 법적 책임은 판매자에게 귀속됨을 고지
