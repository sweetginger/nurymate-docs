# 셀익스(SellEX) 해외배송 모듈 연동 기획서

- **문서 버전**: v1.0
- **작성일**: 2026-03-06
- **태스크**: [10-6]

---

## 1. 셀익스 서비스 개요

### 1.1 한 줄 요약

**셀익스(SellEX)**는 해외 판매처 주문 입력 → 배송 → 수출신고를 원스톱으로 처리하는 해외배송 관리 플랫폼이다. 셀메이트(Sellmate) 그룹이 운영하며, 직접 운송·풀필먼트는 수행하지 않고 **플랫폼 간 연결과 서류 자동화에 집중**한다.

### 1.2 주요 기능

| 기능 | 설명 |
|------|------|
| 해외 판매처 주문 자동 수집 | Cafe24 글로벌몰, Rakuten, Qoo10, Shopify, Shopee 등 연동 마켓에서 주문 자동 수집 |
| 원클릭 수출신고 | 배송과 수출신고를 한 번에 처리. 1건부터 수량 제한 없이 지원 |
| 라스트마일 배송 라벨 발행 | 포워더·특송사별 표준 라벨 자동 생성 |
| 통관용 데이터 생성 | HS코드·가격·구성품 정보 기반 통관 서류 자동화 |
| 포워더 API 연계 | 수집·전송·배송 상태 동기화 |
| 운영 어드민 | 접수/전송 결과, 실패 재시도, 이력 조회 |

### 1.3 셀메이트 OMS와의 관계

셀익스 사용자는 셀메이트 서비스를 유료로 사용하지 않더라도, **주문 수집 용도로 셀메이트 OMS 기능을 무료 제공**받는다. 이는 셀메이트CM 연동 설계 시 중요한 포인트다 — 셀메이트CM의 주문을 셀메이트 OMS를 거치거나 셀익스에 직접 전달하는 방식 모두 검토가 필요하다.

## 2. 연동 목적 및 판매자 가치

### 2.1 셀메이트CM 연동의 전략적 목적

셀메이트CM는 한국 → 일본·글로벌 판매를 핵심 시장으로 설정하고 있다. 해외배송 기능은 셀메이트CM가 카페24 대비 갖춰야 할 차별화 포인트이며, 셀익스와의 연동은 이를 빠르게 실현하는 파트너십 모델이다.

셀익스를 직접 구축하는 것이 아니라 **연동 레이어**로 연결함으로써:
- 개발 리소스 절약 (수출신고 시스템은 규제 연계 복잡도가 높음)
- 검증된 포워더 네트워크 즉시 활용
- 판매자가 이미 알고 있는 셀익스 브랜드 신뢰도 활용

### 2.2 판매자 가치 제안

| 판매자 고통점 | 셀익스 연동 후 |
|------------|-------------|
| 해외 주문을 수동으로 별도 시스템에 입력 | 셀메이트CM 주문 발생 시 셀익스에 자동 전달 |
| 수출신고 절차를 모르거나 외주 처리 | 원클릭 수출신고로 처리 시간 대폭 단축 |
| 포워더별로 다른 라벨 형식 수동 작성 | 포워더 선택 시 라벨 자동 생성 |
| 배송 상태를 셀익스 어드민에서 별도 확인 | 셀메이트CM 대시보드에서 통합 조회 |
| 해외배송 진입 장벽 (최소 수량, 복잡한 절차) | 1건부터 수량 제한 없이 해외배송 가능 |

### 2.3 셀메이트CM 비즈니스 가치

- 해외 판매 판매자의 셀메이트CM 선택 동기 강화
- 셀익스 사용 판매자의 셀메이트CM 유입 채널 (셀메이트 생태계 교차 유입)
- 추후 셀익스 유료 기능 연계 수익 분배 구조 검토 기반 마련

## 3. 연동 플로우 설계

### 3.1 셀메이트CM 주문 발생 → 셀익스 자동 연동

**전체 플로우**

```
[해외 구매자]
    │  주문 완료
    ▼
[셀메이트CM 주문 시스템]
    │  주문 상태: "결제 완료"
    │
    ├─ [국내 배송 주문] → 기존 배송 프로세스
    │
    └─ [해외 배송 주문] → 셀익스 연동 대상
           │
           ▼
    [연동 활성화 여부 확인]
       ├─ 비활성화: 판매자 어드민에 셀익스 연동 안내 배너 노출
       └─ 활성화:
              │
              ▼
       [주문 데이터 변환]  ← 셀메이트CM 주문 스키마 → 셀익스 API 포맷
              │
              ▼
       [셀익스 API 전송]  POST /orders
              │
              ├─ 성공: 셀익스 주문 ID 저장 → 셀메이트CM 주문에 연결
              └─ 실패: 오류 로그 + 판매자 알림 + 재시도 큐 등록
```

**연동 트리거 조건**

| 조건 | 처리 |
|------|------|
| 배송지 국가 = 한국 | 셀익스 전송 안 함 |
| 배송지 국가 ≠ 한국 + 셀익스 연동 활성화 | 셀익스 자동 전송 |
| 배송지 국가 ≠ 한국 + 셀익스 연동 비활성화 | 판매자 어드민에 안내 배너 |
| 부분 취소·환불 발생 | 셀익스에 취소/수정 요청 전송 |

### 3.2 수출신고 → 배송 추적 → 완료 상태 동기화

**상태 동기화 흐름**

```
[셀익스 내부 처리]                    [셀메이트CM 동기화]

주문 접수 완료          ──Webhook──▶  주문 상태: "해외배송 준비 중"
수출신고 완료           ──Webhook──▶  주문 상태: "수출신고 완료"
라벨 발행 완료          ──Webhook──▶  주문 상태: "배송 라벨 발행 완료"
포워더 픽업 완료        ──Webhook──▶  주문 상태: "배송 중"
현지 배송 추적 업데이트  ──Webhook──▶  배송 추적 정보 업데이트 (구매자 노출)
배송 완료               ──Webhook──▶  주문 상태: "배송 완료"
반송/분실 발생          ──Webhook──▶  주문 상태: "배송 이슈" + 판매자 알림
```

**상태 코드 매핑 테이블**

| 셀익스 상태 코드 | 셀메이트CM 주문 상태 | 구매자 노출 문구 |
|---------------|-----------------|--------------|
| `RECEIVED` | `overseas_received` | 해외배송 준비 중 |
| `EXPORT_DECLARED` | `export_declared` | 수출신고 완료 |
| `LABEL_ISSUED` | `label_issued` | 배송 라벨 발행 완료 |
| `PICKED_UP` | `in_transit` | 배송 중 |
| `IN_TRANSIT` | `in_transit` | 배송 중 (현지 추적 중) |
| `OUT_FOR_DELIVERY` | `out_for_delivery` | 배달 중 |
| `DELIVERED` | `delivered` | 배송 완료 |
| `RETURNED` | `return_in_progress` | 반송 처리 중 |
| `LOST` | `delivery_issue` | 배송 이슈 (고객 센터 문의 안내) |

**Webhook 수신 처리**

- 엔드포인트: `POST /api/v1/webhooks/sellex`
- 인증: HMAC-SHA256 서명 검증 (셀익스 Secret Key 기반)
- 멱등성 보장: `sellex_event_id` 기준 중복 처리 방지
- 실패 재시도: 셀익스 측 3회 재전송 (5분·15분·1시간 간격)
- 수신 실패 시: 셀메이트CM 측 폴링 보조 (1시간 주기)

## 5. 셀익스 미사용 판매자 온보딩

### 5.1 연동 활성화 유도 UX

셀익스 연동을 아직 설정하지 않은 판매자가 해외 주문을 받았을 때, 셀메이트CM가 적극적으로 셀익스 연동을 안내한다.

**트리거 시나리오 1: 첫 해외 주문 발생**

```
[알림 배너 — 주문 관리 페이지 상단]

🌏 해외 주문이 도착했습니다!
일본(도쿄) 배송 주문 #10234가 결제 완료되었습니다.

셀익스(SellEX)를 연동하면 수출신고·라벨 발행을 원클릭으로 처리할 수 있습니다.
1건부터 수량 제한 없이 해외배송이 가능합니다.

[셀익스 연동하기 →]   [나중에]
```

**트리거 시나리오 2: 설정 > 해외배송 탭 진입**

셀익스 연동 설정 페이지 상단에 "셀익스란?" 소개 섹션 노출:

```
[셀익스(SellEX) 해외배송 플랫폼]

해외 판매처 주문 수집 → 배송 → 수출신고를 원스톱으로 처리합니다.
- 1건부터 수량 제한 없이 해외배송
- 원클릭 수출신고 (복잡한 세관 절차 자동화)
- CJ대한통운, EMS, DHL, FedEx 등 포워더 연계
- 셀메이트CM 연동 시 주문 자동 전송

[셀익스 무료 시작하기 →]  (새 탭: sellmate.io/sellEX)
셀익스 계정이 있으신가요? → [API 키 입력하여 연동하기]
```

### 5.2 셀익스 가입 안내 플로우

셀익스 계정이 없는 판매자를 위한 가입 안내 플로우.

```
[Step 1] 셀메이트CM 어드민에서 "셀익스 연동하기" 클릭
    │
    ▼
[Step 2] 셀익스 소개 모달 (셀메이트CM 내)
  - 주요 기능 3가지 강조 (수출신고 원클릭 / 1건부터 가능 / 주요 포워더 연계)
  - "셀익스 가입하기 →" 버튼 (sellmate.io 새 탭 오픈)
    │
    ▼
[Step 3] 셀익스 가입 완료 후
  - 셀익스 어드민 → API 키 발급
    │
    ▼
[Step 4] 셀메이트CM로 복귀
  - 설정 페이지에서 API 키 입력 → 검증 → 저장
    │
    ▼
[Step 5] 연동 완료 확인
  - "셀익스 연동이 완료되었습니다. 다음 해외 주문부터 자동으로 처리됩니다." 안내
```

**가입 유도 메트릭**

- 해외 주문 발생 후 셀익스 연동 페이지 방문율
- 방문 → 연동 완료 전환율
- 연동 완료 후 첫 셀익스 자동 전송 성공률

## 6. 국가별 지원 포워더·특송사 목록 노출

### 6.1 포워더 선택 UI

주문 발송 처리 시, 해당 배송 국가에서 셀익스가 지원하는 포워더·특송사 목록을 표시한다.

```
[발송 처리 — 주문 #10234]

배송 국가: 일본 (JP)
예상 배송지: 도쿄도 시부야구

[포워더 선택]
───────────────────────────────
  ● CJ대한통운 국제특송   평균 3~5일  ¥1,200~
    (K-Packet, EMS 포함)
  ○ EMS (우편)            평균 4~7일  ¥900~
  ○ DHL Express           평균 2~3일  ¥3,500~
```

### 7.2 셀익스 → 셀메이트CM: 상태 업데이트 (Webhook)

**엔드포인트 (셀메이트CM 수신)**

```
POST https://api.sellmatecm.com/api/v1/webhooks/sellex
X-Sellex-Signature: sha256={HMAC_SIGNATURE}
Content-Type: application/json
```

**페이로드 예시**

```json
{
  "event_id": "evt-sellex-abc123",
  "event_type": "order.status_updated",
  "occurred_at": "2026-02-21T14:30:00Z",
  "data": {
    "sellex_order_id": "SEX-2026-00234",
    "external_order_id": "nurymate-10234",
    "status": "PICKED_UP",
    "tracking_number": "JX123456789JP",
    "forwarder": "CJ_INTL",
    "tracking_url": "https://trace.cjlogistics.com/JX123456789JP"
  }
}
```

**셀메이트CM 처리 로직**

1. HMAC 서명 검증 → 실패 시 401 반환
2. `event_id` 중복 체크 → 중복 시 200 반환 (멱등성)
3. `external_order_id` 기준 셀메이트CM 주문 조회
4. 주문 상태 업데이트 + 추적 정보 저장
5. 구매자 알림 발송 (이메일·SMS) — 해당되는 경우
6. 응답: `200 OK`

### 7.3 셀메이트CM → 셀익스: 주문 취소·수정

**취소 요청**

```
DELETE https://api.sellmate.io/sellex/v1/orders/{sellex_order_id}
```

**수량·수령자 수정 요청**

```
PATCH https://api.sellmate.io/sellex/v1/orders/{sellex_order_id}
```

취소·수정은 셀익스에서 수출신고 또는 라벨 발행이 완료된 이후에는 불가능하다. 해당 상태에서의 취소는 셀익스 어드민에서 수동 처리해야 하며, 셀메이트CM는 판매자에게 이를 안내한다.
