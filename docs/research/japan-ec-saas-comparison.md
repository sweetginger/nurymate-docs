# 일본 D2C EC SaaS 비교 조사 (BASE·STORES 등) [4-14]

> 조사일: 2026-03-10 | 목적: 누리메이트 일본 시장 차별화 전략 수립

---

## 1. 일본 EC SaaS 시장 개요

일본의 EC 시장 규모는 2025년 기준 약 2,865억 달러(약 43조 엔)로, 2034년까지 연평균 10.15% 성장하여 7,018억 달러 규모에 달할 것으로 전망된다. 이 중 D2C(Direct-to-Consumer) 시장은 SaaS형 EC 플랫폼의 성장과 함께 확대되고 있으며, 일본 내 주요 플레이어는 BASE·STORES·카라미숍(カラーミーショップ)·Makeshop·futureshop 등이 있다.

일본 D2C EC SaaS는 크게 두 세그먼트로 나뉜다:
- **프리미엄 무료 진입형:** BASE, STORES — 초기 비용 없이 시작, 판매 수수료 부과 구조
- **월정액 기능형:** 카라미숍, Makeshop, futureshop — 월정액 지불 후 낮은 수수료 구조

## 2. 플랫폼별 핵심 특징 및 포지셔닝

### 2-1. BASE (ベイス)

**설립:** 2012년 | **운영사:** BASE 주식회사 | **점포 수:** 약 200만 개 이상 (2024년 기준)

**포지셔닝:** 일본에서 가장 많은 점포 수를 보유한 소규모·개인 셀러 대상 플랫폼. "誰でも簡単にネットショップが開ける(누구나 쉽게 쇼핑몰을 열 수 있다)"를 모토로 진입 장벽을 극단적으로 낮춘다.

**타겟 고객:**
- 개인 크리에이터·핸드메이드 작가
- 소규모 패션·잡화 브랜드
- 부업·사이드 비즈니스 운영자

**핵심 특징:**
- 초기 비용·월정액 0원
- 2분 만에 쇼핑몰 개설 가능
- 앱(플러그인) 마켓을 통한 기능 확장 구조
- BASE 자체 집객 기능(BASE 검색, 팔로워 관리)
- 스마트폰 앱으로 관리 가능

**약점:**
- 판매 수수료가 높아 매출이 늘수록 비용 증가
- SEO·마케팅 도구 빈약
- B2B·도매 기능 없음
- 글로벌 판매 제한적

---

### 2-2. STORES (ストアーズ)

**설립:** 2012년 | **운영사:** STORES 주식회사 | **점포 수:** 약 140만 개 이상 (2024년 기준)

**포지셔닝:** 온라인 쇼핑몰과 실점포 POS를 통합 관리하는 "OMO(Online Merges with Offline)" 플랫폼. 2025년 3월 신규 통합 플랜 출시로 온라인몰·캐시리스 결제·POS 레지·예약 관리 등 7가지 서비스를 월 3,300엔으로 묶음 제공.

**타겟 고객:**
- 실점포와 온라인몰을 동시 운영하는 소매업자
- 카페·레스토랑·소규모 브랜드
- 예약·티케팅이 필요한 사업자 (STORES 예약 연동)

**핵심 특징:**
- 실점포 POS와 재고 일원 관리
- STORES 결제 (캐시리스 결제 단말) 연동
- 예약 시스템(STORES 予約) 연동
- 2025년 신플랜: 월 0엔 + 캐시리스 결제 수수료 2.48%~

**약점:**
- 쇼핑몰 디자인 커스터마이징 제한적
- 글로벌·크로스보더 판매 미지원
- 고급 마케팅 자동화 기능 부족

---

### 2-3. 카라미숍 (カラーミーショップ)

**운영사:** GMO 페파보 (GMO Pepabo) | **점포 수:** 약 18만 개

**포지셔닝:** GMO 그룹의 인프라를 기반으로 중소 규모 EC 사업자를 타겟으로 한 안정적·기능 충실형 플랫폼. 전화·이메일 지원, 세미나 개최 등 밀착 지원이 강점.

**요금제:**
| 플랜 | 초기 비용 | 월정액 | 결제 수수료 |
|------|----------|--------|-------------|
| 무료 | 0엔 | 0엔 | 6.6% + 결제 수수료 |
| 레귤러 | 3,300엔 | 4,950엔 | 3.19% + 결제 수수료 |
| 라지 | 3,300엔 | 9,900엔 | 3.19% + 결제 수수료 |
| 프리미엄 | 3,300엔 | 39,600엔 | 3.19% + 결제 수수료 |

**크로스보더 지원:**
- 월드 쉬핑 BIZ 앱(World Shipping BIZ for カラーミーショップ) 연동 시 228개국 해외 배송 가능
- Buyee Connect 앱으로 해외 구매대행 서비스 연동 가능

**약점:**
- 기본 다국어·다통화 지원 없음 (앱 연동 필요)
- 디자인 자유도가 Shopify보다 낮음

---

### 2-4. Makeshop

**운영사:** GMO 메이크숍 (GMO MakeShop) | **점포 수:** 약 11,000개

**포지셔닝:** 651개 기능을 보유한 기능 최강의 EC 플랫폼. 중견~대형 EC 사업자를 타겟으로 하며, 낮은 결제 수수료로 매출 규모가 클수록 유리하다.

**요금제:**
| 플랜 | 초기 비용 | 월정액 | 결제 수수료 |
|------|----------|--------|-------------|
| 스탠다드 | 11,000엔 | 11,000엔 | 3.19% |
| 프로페셔널 | 11,000엔 | 22,000엔 | 3.19% |
| 엔터프라이즈 | 11,000엔~ | 55,000엔~ | 별도 협의 |

**주요 강점:**
- 국내 EC SaaS 중 최다 기능 수 (651개)
- 결제 수수료가 상대적으로 낮음
- 정기 구독·정기 구매 기능 강력
- 대규모 상품 수 관리 용이
- 전화 지원 포함

**약점:**
- 초기 비용·월정액이 높아 소규모 사업자에게 부담
- 글로벌 판매 기능은 별도 설정 필요
- UI가 다소 복잡

---

### 2-5. futureshop

**운영사:** 주식회사 futureshop | **점포 수:** 약 2,700개

**포지셔닝:** 중견~중대형 패션·라이프스타일 브랜드의 자사 EC 사이트를 전문으로 지원. 높은 디자인 자유도와 OMO 기능 강점.

**요금제:** 초기 비용 22,000엔~, 월정액 24,000엔~

**주요 강점:**
- 디자인 자유도가 높음 (헤드리스 구성 가능)
- 오프라인 매장과의 OMO 연동 기능
- 재고 관리·CRM 기능 강력
- 전담 컨설팅 지원

**약점:**
- 비용이 높아 소규모 사업자 진입 장벽
- 점포 수가 적어 커뮤니티 생태계 빈약
- 글로벌 기능 제한적

## 4. 강점·약점 및 사용자 평가

### BASE 사용자 평가 (일본어 커뮤니티 기반)
- 긍정: "즉시 시작할 수 있다", "스마트폰으로 관리 가능", "디자인 템플릿이 예쁘다"
- 부정: "수수료가 너무 비싸다 — 팔면 팔수록 손해", "SEO가 약하다", "고객 지원이 느리다"

### STORES 사용자 평가
- 긍정: "실점포와 통합 관리가 편리", "POS 연동이 매끄럽다"
- 부정: "디자인 커스터마이징이 너무 제한적", "해외 배송 설정이 복잡"

### 카라미숍 사용자 평가
- 긍정: "전화 지원이 있어 안심", "기능이 충분히 많다"
- 부정: "관리 화면이 구식", "앱 연동이 복잡"

### Makeshop 사용자 평가
- 긍정: "기능이 가장 많다", "결제 수수료가 낮아 고매출 사업자에게 유리"
- 부정: "UI가 복잡", "초기 비용과 월정액이 높다"

## 7. 누리메이트 일본 시장 차별화 전략 시사점

### 7-1. 일본 EC SaaS의 공통 약점 (누리메이트의 기회)

1. **크로스보더 지원 부재:** 일본 내수 플랫폼 전체가 크로스보더(한국→일본, 일본→해외)를 기본 지원하지 않는다. 누리메이트가 한국-일본 크로스보더에 특화된다면 즉각적인 차별화 가능.

2. **K-콘텐츠·K-브랜드 특화 부재:** BASE·STORES 등은 K-뷰티·K-패션 판매자를 위한 전용 기능(한국 PG 연동, 한국어 관리 화면, 한국 물류 연동)이 없다.

3. **배송 연동 UX 열악:** BASE·STORES는 콘비니 수령, 야마토 API 연동 등을 기본 지원하지 않는다. 누리메이트가 야마토 API를 내장하면 "배송까지 완성된 플랫폼"으로 포지셔닝 가능.

4. **마이그레이션 도구 부재:** 라쿠텐·아마존 JP에서 D2C로 이전하려는 셀러를 위한 상품 데이터 마이그레이션 기능이 시장에 존재하지 않는다.

### 7-2. 포지셔닝 전략 제언

| 타겟 세그먼트 | 현재 사용 플랫폼 | 누리메이트 이전 유인 |
|-------------|----------------|-------------------|
| K-브랜드 일본 진출 희망 기업 | 카페24 (국내) | 일본 특화 배송·결제·콘비니 연동 기본 내장 |
| 라쿠텐·아마존 셀러 (D2C 전환 희망) | 라쿠텐/아마존 | 상품 데이터 마이그레이션 자동화 + 수수료 절감 |
| 일본 소규모 패션·뷰티 브랜드 | BASE / STORES | 더 낮은 수수료 + 크로스보더 기능 |

### 7-3. 핵심 차별화 포인트

- **"일본 배송 완성형":** 야마토 API 내장 → 콘비니 수령 선택 UI 기본 제공 (BASE·STORES 미지원)
- **"라쿠텐 졸업 플랫폼":** 라쿠텐·아마존 상품 데이터 원클릭 마이그레이션
- **"한국-일본 원스탑":** 한국 관리자, 일본 고객, 양국 물류 동시 지원
- **요금 투명성:** 숨겨진 수수료 없이 단순한 월정액 모델
