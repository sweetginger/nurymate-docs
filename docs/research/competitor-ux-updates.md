## 조사 결과

### 카페24

#### 최근 주요 업데이트 (2022~2026)

- **2022년**: 에디봇 `AI 이미지 크롭` 출시 — 쇼핑몰 최근 이미지 비율을 자동 조회·크롭, 모델 얼굴 인식 후 최적 편집. `AI 이미지 자동 분류` 추가 — 업로드 이미지를 모델컷/색상컷/디테일컷으로 자동 분류·배치. 에디봇 전시관 공개(70개+ 템플릿), 누적 상세페이지 제작 500만 회 돌파
- **2023년**: 리소스 잠금 아이콘 추가(삭제 방지). 에디봇 소셜(SNS 통합 콘텐츠 관리) 출시
- **2024년 2월**: AI 파트너사 42개사 제휴 — 이커머스 AI 기능 생태계 확장. 와들 'Gentoo(젠투)' 구매 전환 AI 에이전트 연동 포함
- **2024년**: 고객 정보 CSV 일괄 불러오기 추가. 카페24 허브 WMS 연동(셀메이트 없이도 연동). AI 앱 스토어 거래액 679억 원(AI 활용 앱 비중 약 20%)
- **2025년 3월**: 주문서 추가항목 API 신규 추가. HS코드 수정·조회. 결제수단 표기방식(텍스트/로고 아이콘) 선택 기능 개선
- **2025년 하반기**: AEO/GEO 시대 대응 SEO 고도화 — 언어별 메타데이터 자동 최적화, hreflang 자동 관리. 사이트 구조·브랜드 스토리·상품 카탈로그를 AI 친화 형식으로 자동 생성
- **2026년 2월**: 국내 이커머스 플랫폼 최초 오픈AI '앱스 인 챗GPT' 전용 앱 출시 — 챗GPT 내에서 카페24 스토어 상품 추천·비교·구매 가능

#### 에디봇 AI 기능 전체 목록 (2026년 기준)

| 기능 | 설명 |
|------|------|
| AI 상세페이지 자동 생성 | 이미지 업로드 → AI가 카테고리 인식 → 템플릿 자동 적용 → 상세페이지 초안 생성. 100분→3분 |
| AI 이미지 자동 분류 | 업로드 이미지를 모델컷/색상컷/디테일컷으로 자동 분류·배치 |
| AI 이미지 크롭 | 쇼핑몰 비율 자동 감지 + 모델 얼굴 인식 기반 최적 썸네일 편집 |
| AI 상품 카테고리 인식 | 이미지에서 카테고리·색상 자동 인식 |
| 에디봇 배너 | 쇼핑몰 배너 AI 자동 제작 |
| 에디봇 소셜 | SNS 통합 관리 + SNS 투고 콘텐츠 제작 지원 |
| 에디봇 프로모션 | 이벤트 등록~통계 분석 일원화 |
| 에디봇 리뷰 | 고객 상품 후기를 상세페이지에 자동 삽입 |
| 에디봇 전시관 | AI 기반 쇼핑몰 기획전·카테고리 전시 자동 구성 |

> ⚠️ 기존 문서에 기재된 "SNS 투고문·문의 대응문 자동 생성" 기능은 공식 자료에서 명시 미확인. 에디봇 소셜이 SNS 콘텐츠를 지원하나 LLM 기반 문장 생성 여부는 불명확.

#### 사용자 편의 개선 핵심 항목

- 에디봇 AI로 상세페이지 제작 100분 → 3분 단축
- AI 이미지 분류·크롭 자동화 (편집 공수 제거)
- 챗GPT 앱 연동으로 AI 플랫폼 내 쇼핑 채널화 (AEO 대응)
- 42개 AI 파트너 생태계로 기능 확장 가속

#### 셀메이트CM 대비 현황

카페24는 에디봇(이미지·콘텐츠 AI)과 AI 생태계 파트너십에서 2022년부터 앞서 있으나, AI 스토어 설정 자동화·온보딩 마법사 미보유. 챗GPT 채널 연동은 셀메이트CM의 AEO/GEO 전략과 겹치는 영역으로 선점 속도 주시 필요. 일본 결제·크로스보더는 별도 앱 연동 구조로 셀메이트CM 네이티브 지원 대비 열위. 통합 정산 대시보드 미제공.

### 아임웹

#### 최근 주요 업데이트 (2022~2025)

- **2024년 4월**: **주문관리시스템(OMS) 전면 리뉴얼** — 공식 명칭 '주문관리 v2'. 2024년 4월 24일부터 순차 적용, 6월 전체 완료. 하기 OMS 상세 항목 참조
- **2024년 5월**: CRM 캠페인 자동화 출시 — 회원가입 후 1일 미구매, 장바구니 1시간 방치, 1개월 재구매 없음 등 행동 트리거별 카카오톡 메시지·쿠폰 자동 발송. 출시 1개월 내 구매 전환율 평균 85% 향상 측정
- **2024년 8월**: '전문가 찾기' 서비스 출시 — 쇼핑몰 제작·운영 전문가와 셀러를 연결하는 마켓플레이스
- **2025년**: AI 사이트 빌더 '아임AI' 출시. AI 기술지원 챗봇 'BIBI(비비)' 출시. 누적 개설 사이트 100만 개·누적 거래액 6조~7조 원 돌파

#### OMS 전면 리뉴얼 상세 (주문관리 v2, 2024년 4월)

**1. 주문 목록 통합**
- 기존: 결제 상태·배송 상태별로 메뉴 분산 → 리뉴얼: 주문 메뉴 하나에서 전체 조회 가능
- 세분화된 검색 필터, 대량 주문 일괄 처리 지원

**2. 타임라인 이력 관리**
- 주문별 전체 처리 이력(접수→결제→배송→완료)을 시간 순으로 시각화
- 담당자 메모 기능 포함

**3. 추가 결제 요청 (아임웹 독자 기능)**
- UX 흐름: 관리자가 품목·수량 수정 → 추가 금액 산출 → "고객 마이페이지로 추가 결제 요청" 클릭 → 고객 알림 발송 → 고객이 마이페이지에서 직접 결제 완료
- 기존 방식(가상계좌 안내, 별도 링크 생성)을 완전 대체. 카페24·고도몰은 이 흐름 미지원

**4. 부분 처리 강화**
- 하나의 주문에서 일부 품목·수량만 취소·반품·교환 가능
- 교환 승인 시 동일 상품 자동 추가(상품 준비 상태)로 전환 → 기존 배송 흐름으로 진행
- 부분 취소·반품 시 변동 금액은 마이페이지 추가 결제 요청 기능 연동

**5. 취소·환불 흐름 표준화**
- [취소 접수 → 취소 승인 → 환불] 3단계 표준화
- 자동 환불: 카드결제·카카오페이 등 PG 취소 승인 시 자동 처리
- 수동 환불: 무통장입금·가상계좌·휴대폰 결제는 관리자 수동 이체 후 처리

#### 아임AI — AI 사이트 자동 생성

- **UX 흐름**: 콘셉트 텍스트 입력 → 사이트명·URL 입력 → AI 생성(약 3분 소요) → 미리보기 → 공개 → 요금제 선택(14일 무료 체험)
- 생성 후 어드민에서 모든 디자인·콘텐츠 자유 편집 가능
- **제약**: 하루 1회 생성 제한. 생성 중 페이지 이탈 시 사이트 삭제. 프롬프트 품질에 따라 결과 차이 큼

#### BIBI(비비) — AI 기술지원 챗봇 상세

- **기술 구조**: 채널톡 기반 ❌ — **아임웹 자체 개발**. 1,300여 개 아임웹 공식 기술 문서 기반 RAG(검색 증강 생성) 방식으로 구현
  - 외부 범용 챗봇 도입 시 "정확히 매칭되는 질문만 답변 가능"한 한계를 극복하기 위해 자체 문서 탐색 방식 채택
  - 사용 LLM 모델(GPT-4, Claude 등)은 공식 공개 미확인
- **접속**: bb.imweb.me — 아임웹 계정 로그인 후 이용
- **기능 범위**: 아임웹 사용법 질문 답변 (어드민 기능, 디자인 모드, 주문 관리 방법 등). 자연어로 질문하면 즉시 답변
- **중요 한계**: BIBI는 **아임웹 운영자(셀러) 대상 기술지원 챗봇**임. 쇼핑몰 방문 고객 대상 CS 챗봇이 아님. 고객 응대용 챗봇은 채널톡 등 외부 앱을 별도 설치해야 함

#### 사용자 편의 개선 핵심 항목

- OMS 마이페이지 추가 결제 요청으로 주문 수정 후속 결제 마찰 제거 (아임웹 독자 기능)
- CRM 자동화로 구매 유도 메시지 자동 처리 (전환율 +85%)
- 아임AI로 비개발자 대화형 쇼핑몰 생성 (~3분)
- BIBI로 어드민 사용법 문의 AI 즉시 응대 (CS 공수 절감)

#### 셀메이트CM 대비 현황

아임웹 OMS의 마이페이지 추가 결제 요청은 카페24·고도몰에 없는 독자 기능으로 셀메이트CM 주문 관리 기획 시 참고 필요. BIBI의 자체 RAG 방식은 셀메이트CM M-09 AI 채팅의 기술적 방향과 유사. 아임웹은 한국 단일 시장 기반으로 일본 결제·다국어 지원 없음 — 이 점이 셀메이트CM의 핵심 차별점.

### 고도몰

#### 최근 주요 업데이트 (2022~2025)

- **2024년 2월**: 상품 등록·수정 성능(속도) 대폭 개선 패치 — 상품 대량 관리 시 응답 속도 개선
- **2024년 6월**: 대규모 업데이트 단행 (NHN커머스 주도)
  - 인프라: 온프레미스(On-Premise) → SaaS(클라우드) 전환으로 성능·보안·비용 효율 개선
  - 기능 내재화: 간편 로그인, 공급사 관리, 사은품 관리, 프로모션 등 사용 빈도 높은 부가 기능 11개를 플러스샵 앱에서 기본 관리자 메뉴로 이전
  - 간편 로그인 메뉴 경로 개선: `회원 > 소셜로그인` → `회원 > 간편로그인` (경로 단순화)
  - 이전센터·튜닝센터 오픈 (타 솔루션→고도몰 이전 전담 지원)
- **2024년 12월**: 무료형 요금제 '고도몰 basic' 출시
- **2025년 4월**: 요금제 라인업 개편 (basic / pro 이분화)
- **2025년 5월**: 채널톡 간편 설치 앱 출시 — 앱스토어 원클릭 채널톡 설치 지원

#### 사용자 편의 개선 핵심 항목

- 자주 쓰는 11개 기능 기본 메뉴 내재화 (앱 추가 설치·결제 불필요)
- 간편 로그인 메뉴 구조 단순화
- 클라우드 전환으로 속도·안정성 향상 (운영 안정감)
- 무료 요금제 신설로 진입 장벽 제거

#### 셀메이트CM 대비 현황

레거시 온프레미스에서 SaaS로의 전환이 2024년에야 완료된 상태로 기술 부채가 상당함. 고도몰 basic 무료 요금제는 셀메이트CM의 Starter 0% 수수료 전략과 유사한 방향이나, 고도몰은 국내 시장 한정. 온보딩 자동화·AI 기능 미흡.

- **결제**: 엑심베이(Eximbay)를 통해 해외 PG 연동. **라인페이(LINE Pay) 지원** (일본 고객 대상 주력). 신용카드·해외 PG 기본 포함. 환율 설정 기능 제공
  - 콘비니 결제 지원 여부: 공식 공개 자료에서 확인 불가 (직접 문의 필요)

- **요금**: 공개 자료에서 메이크글로비 별도 요금 명시 없음 (직접 문의 필요). 메이크샵 기본 요금(무료 프리 / 월 5.5만 원 프리미엄)과 별도 구조로 추정

메이크샵의 스마트스토어 원클릭 전환은 네이버 생태계 내 이전에 특화됨. 셀메이트CM의 카페24 마이그레이션과 방향은 유사하나 대상 플랫폼이 다름. 메이크글로비가 일본·글로벌을 지원하나 별도 솔루션 계약 구조이고 콘비니·PayPay 등 일본 핵심 결제 네이티브 지원은 불확실. 셀메이트CM의 한·일 결제 네이티브 통합이 차별점으로 유효.

## 핵심 인사이트

1. **AI 기능이 경쟁의 중심축으로 완전히 이동했다.** Shopify(Sidekick·Magic·AI 스토어 빌더), 아임웹(아임AI·BIBI), BASE(AIアシスタント), 카페24(에디봇 확장), 메이크샵(AI 상품 설명)까지 7개 플랫폼 전부가 2023~2025년 사이 AI 기능을 핵심 업데이트로 밀고 있다. 셀메이트CM가 AI 온보딩·AI 프론트빌더를 P0으로 배치한 방향은 맞으나, 출시 직후부터 이 기능이 실제로 작동해야만 "AI 플랫폼"이라는 포지셔닝이 유효하다.

2. **소셜 커머스 연동이 급속히 표준 기능화되고 있다.** BASE의 TikTok Shop API 연동(2025.7), YouTube 쇼핑 연동(2024.5), 카페24의 SNS 연동 앱 생태계 확장 등 판매 채널이 자사몰 단독에서 SNS 내 구매로 확장되는 속도가 빠르다. 셀메이트CM의 SNS 마케팅 연동(P1)을 런칭 직후 6개월 내 구현하지 않으면 경쟁력 격차가 벌어진다.

3. **정산 속도와 어드민 분석이 셀러 락인의 핵심 변수로 부상했다.** BASE의 최속 10분 정산, STORES의 통합 데이터 분석, 카페24의 대시보드 강화는 모두 "돈이 빠르게 들어오고, 데이터로 판단할 수 있는" 운영 편의를 강화하는 방향이다. 셀메이트CM의 통합 결제 대시보드(P1)와 즉시 정산 구조 설계는 경쟁 플랫폼 대비 늦지 않게 Phase 2 내 구현이 필요하다.
