    글로벌 인플루언서 DB 수집 및 중개 기능 기획서¶

    문서 버전: v1.0 작성일: 2026-03-06 관련 태스크: [11-1]

    1. 전략 배경 및 목표¶

    1.1 배경¶

    패션 D2C 판매자의 가장 큰 고민 중 하나는 마케팅 채널 확보다. 특히 초기 판매자는 인플루언서를 직접 발굴하고 컨택·협상·계약·성과 측정까지 혼자 감당해야 한다. 이 과정에서 발생하는 마찰(friction)이 크다.

    • 인플루언서 직접 컨택 → 사기·미이행 리스크 발생
    • 국가별 컨택 채널 상이 (DM, 이메일, 에이전시 등)
    • 협업 결과물 품질·납기 관리 어려움
    • 성과 측정 기준 없이 돈만 지출하는 구조

    셀메이트CM은 이 마찰을 플랫폼 내부에서 해소하는 것으로 판매자 락인(Lock-in)을 달성할 수 있다. 인플루언서 마케팅 기능이 플랫폼에 내장되면 판매자는 셀메이트CM를 이탈하기 어려워진다.

    1.2 시장 근거¶

    • 글로벌 인플루언서 마케팅 시장 규모: 2024년 240억 달러 → 2025년 330억 달러 이상 추정 (Statista)
    • 인플루언서 마케팅 플랫폼 주요 사례:
    • Modash: 350만 명 이상 인플루언서 DB (Instagram, YouTube, TikTok), 팔로워 1,000명 이상 전수 수집
    • Aspire (AspireIQ): 이커머스 특화, UGC 라이선싱·어필리에이트 트래킹 통합
    • GRIN: Shopify 연동에 최적화, 캠페인 전체 생애주기 관리
    • Shopify Collabs: Shopify 자체 인플루언서 기능. 무료 설치, 어필리에이트 링크 자동 생성, 수수료 자동 정산 (2.9% 처리 수수료). 셀메이트CM이 이 수준 이상의 기능을 내장해야 경쟁력 확보 가능.

    1.3 목표¶

    목표 지표 달성 시점
    플랫폼 내 인플루언서 생태계 구축 등록 인플루언서 수 10만 명+ Phase 2 (서비스 출시 후 12개월)
    판매자 마케팅 채널 내재화 인플루언서 기능 사용 판매자 비율 30%+ Phase 2
    판매자 이탈율 감소 기여 인플루언서 기능 사용 판매자 이탈율 50% 이하 Phase 3
    수수료 수익 발생 중개 거래액 기준 월 1억 원+ Phase 3

    2. 인플루언서 DB 수집 시스템¶

    2.1 수집 소스 및 API 활용 방안¶

    Instagram Graph API¶

    • 수집 가능 범위 (2025년 기준):
    • 비즈니스·크리에이터 계정이 앱을 직접 인증(OAuth)한 경우에만 접근 가능
    • 인증된 계정의 팔로워 수, 도달 범위(reach), 노출(impressions), 인게이지먼트, 인구통계(상위 45개 세그먼트까지) 수집 가능
    • 팔로워 100명 미만 계정은 팔로워 지표 제한 (2025년 신규 제약)
    • 비공개 크롤링 불가: 공식 API 이외 수집은 Meta ToS 위반
    • 셀메이트CM 활용 전략:
    • 인플루언서가 셀메이트CM 앱을 인증하면 자동으로 데이터 수집·갱신
    • 인증 인센티브 설계 필요 (아래 6장 참조)
    • 제3자 데이터 제공업체(Phyllo, Data365 등) API 병행 활용으로 미인증 계정 기본 정보 보완

    TikTok Creator Marketplace API¶

    • 수집 가능 범위 (2025년 기준):
    • TikTok Creator Marketplace 파트너사(공식 승인)만 접근 가능
    • 크리에이터 인구통계, 성장 추세, 인기 영상, 실시간 캠페인 성과(조회·좋아요·공유·댓글) 수집
    • 2025년 승인 절차 강화: 사용 목적 상세 제출, 심사 기간 수 주 소요
    • 셀메이트CM 활용 전략:
    • 파트너사 자격 신청 (마케팅 플랫폼 카테고리로 신청)
    • 승인 전까지는 크리에이터 자발적 정보 입력 + 공개 데이터 수집으로 대체
    • TikTok Business API 별도 연동하여 광고 캠페인 성과 측정 지원

    YouTube Data API¶

    • 수집 가능 범위 (2025년 기준):
    • 채널 구독자 수, 영상 조회수, 좋아요·댓글 수: OAuth 없이도 공개 채널 기준 수집 가능
    • 상세 시청자 분석(성별·연령·지역): 채널 소유자 OAuth 인증 후 YouTube Analytics API 이용
    • YouTube Shorts 분석, YouTube Shopping 연동 데이터: 2025년 이후 확장 중
    • 셀메이트CM 활용 전략:
    • 공개 채널 데이터는 YouTube Data API v3 (무료 쿼터 내)로 기본 수집
    • 상세 데이터는 인플루언서 인증 시 Analytics API 추가 수집
    • 패션·뷰티 카테고리 유튜버 우선 인덱싱

    제3자 데이터 제공업체 (보완 소스)¶

    업체 특징 활용 방안
    Modash 350만+ 크리에이터, 공개 데이터 기반 초기 DB 구축용 라이선스 계약 검토
    Phyllo (getphyllo.com) 인플루언서 인증 연동 API, 다플랫폼 지원 인플루언서 자발적 연동 시 데이터 집계 인프라
    Data365 Instagram Analytics API 대안 미인증 계정 기본 지표 보완
    HypeAuditor 어뷰징 탐지, 진성 팔로워 비율 분석 품질 필터링 레이어

    2.2 수집 데이터 항목¶

    인플루언서 1명당 수집·관리하는 데이터 항목:

    카테고리 항목 출처
    기본 정보 이름/핸들, 국가, 언어, 주요 활동 플랫폼 공개 API / 직접 입력
    규모 지표 팔로워 수 (플랫폼별), 구독자 수 공개 API
    참여율(ER) 게시물당 평균 좋아요·댓글 / 팔로워 × 100 공개 API 계산
    콘텐츠 카테고리 주요 콘텐츠 태그 (패션, 뷰티, 라이프스타일 등) 자동 분류 (NLP 태깅)
    대상 고객 팔로워 연령·성별·국가 분포 OAuth 인증 후 수집
    콘텐츠 업로드 주기 최근 3개월 평균 게시 빈도 공개 API 분석
    협업 이력 과거 협업 브랜드 카테고리, 협업 빈도 콘텐츠 분석 / 직접 입력
    예상 단가 팔로워 수·ER·카테고리 기반 시장가 추정 자체 모델링
    진성도 점수 어뷰징 탐지 (HypeAuditor API 또는 자체 모델) HypeAuditor API
    컨택 포인트 이메일, 에이전시 여부, 선호 컨택 방법 직접 입력 (인플루언서)
    셀메이트CM 협업 이력 완료 캠페인 수, 평점, 납기 준수율 플랫폼 내부 데이터

    2.3 자동 수집 주기 및 갱신 정책¶

    데이터 유형 갱신 주기 방식
    팔로워 수 · 구독자 수 주 1회 (매주 월요일 새벽) API 배치
    최근 게시물 인게이지먼트 주 1회 API 배치
    대상 고객 인구통계 월 1회 OAuth 인증 계정만
    예상 단가 월 1회 재산정 자체 모델
    진성도 점수 월 1회 HypeAuditor API
    협업 이력 (셀메이트CM 내) 실시간 캠페인 완료 이벤트

    DB 삭제 정책: 6개월 이상 활동 없는 인플루언서는 비활성 처리. 12개월 이상 비활성 시 DB에서 제거 (또는 아카이빙).

    3. 판매자-인플루언서 중개 구조¶

    3.1 셀메이트CM 개입 방식¶

    핵심 원칙: 판매자와 인플루언서 간 직접 컨택 차단

    셀메이트CM이 계약 주체로 개입하는 이유: 1. 수익화: 중개 수수료 수취 기반 확보 2. 락인 강화: 외부에서 형성된 관계가 플랫폼을 떠나지 않도록 3. 품질 보증: 계약 불이행, 콘텐츠 품질 미달 시 플랫폼이 책임지고 환불·재협의 4. 데이터 독점: 캠페인 성과 데이터가 셀메이트CM에 축적

    구분 허용 차단
    판매자 → 인플루언서 플랫폼 내 메시지만 외부 SNS DM, 이메일 직접 공유
    계약서 셀메이트CM 표준 계약 외부 계약서 교환
    정산 셀메이트CM 경유 정산 계좌 직접 이체

    인플루언서 컨택 정보 보호: 이메일·전화번호는 셀메이트CM이 보관, 판매자에게 노출하지 않음.

    3.2 캠페인 생성 → 인플루언서 매칭 → 계약 → 성과 측정 플로우¶

    [판매자]
   │
   ├─ 1. 캠페인 생성
   │      - 상품 선택 / 캠페인 목표 (인지도·전환·팔로워)
   │      - 예산 범위 / 희망 콘텐츠 유형 (릴스·쇼츠·이미지·블로그)
   │      - 희망 인플루언서 조건 (국가·카테고리·팔로워 범위)
   │
   ├─ 2. 매칭 추천 (자동)
   │      - 셀메이트CM 매칭 알고리즘이 후보 3~10명 추천
   │      - 판매자는 프로필 열람 (상세 지표 포함) 후 제안 전송
   │
   ├─ 3. 인플루언서 수락·협의
   │      - 인플루언서: 제안 수락 / 조건 수정 요청 / 거절
   │      - 셀메이트CM 메시지 채널 내에서만 소통
   │
   ├─ 4. 계약 체결
   │      - 셀메이트CM 표준 계약서 자동 생성 (금액·납기·콘텐츠 조건)
   │      - 전자서명 (판매자 + 인플루언서)
   │      - 대금 에스크로 예치 (캠페인 완료 후 정산)
   │
   ├─ 5. 콘텐츠 제작·검수
   │      - 인플루언서: 콘텐츠 초안 업로드 (플랫폼 내)
   │      - 판매자: 수정 요청 또는 승인 (최대 2회 수정 가능)
   │      - 최종 승인 후 콘텐츠 게시
   │
   └─ 6. 성과 측정 및 정산
          - 게시 후 7일·30일 성과 자동 수집 (조회수·클릭·전환)
          - 어필리에이트 링크·쿠폰 코드 기반 전환 추적
          - 성과 확인 후 에스크로 정산 (인플루언서 수취액 = 계약금 - 셀메이트CM 수수료)
          - 판매자 어드민에 ROI 리포트 자동 생성

    4. 매칭 알고리즘 설계¶

    4.1 판매자 상품 카테고리 × 인플루언서 팔로워 특성¶

    매칭 점수(Match Score)는 다음 변수의 가중합으로 산출:

    변수 설명 가중치
    카테고리 일치도 판매자 상품 카테고리 ↔ 인플루언서 주요 콘텐츠 카테고리 코사인 유사도 30%
    팔로워 지역 일치도 판매자 목표 판매 국가 ↔ 인플루언서 팔로워 국가 분포 겹침 비율 25%
    참여율(ER) 인플루언서 ER이 카테고리 평균 이상인지 여부 20%
    예산 적합도 판매자 예산 범위 내 예상 단가 해당 여부 15%
    진성도 점수 어뷰징 탐지 점수 10%

    보조 필터: - 셀메이트CM 플랫폼 내 협업 완료 이력 유무 (우선 노출) - 최근 3개월 활동 여부 (비활성 인플루언서 하위 노출) - 판매자가 이전에 제안했으나 거절당한 인플루언서 제외

    4.2 예산 범위 × 예상 ROI 필터링¶

    예상 ROI 산출 공식 (초기 모델):

    예상 ROI = (예상 도달 × 카테고리 평균 전환율 × 상품 평균 객단가) / 캠페인 비용
    • 예상 도달 = 팔로워 수 × ER × 플랫폼 도달 계수 (Instagram: 0.1~0.3, TikTok: 0.3~0.6)
    • 카테고리 평균 전환율: 플랫폼 내 누적 데이터 기반 (초기에는 업계 벤치마크 사용)
    • ROI < 1.0인 조합은 경고 표시 (판매자가 확인 후 진행 가능)

    예산 티어별 추천 인플루언서 유형:

    예산 인플루언서 유형 팔로워 규모
    10~50만 원 나노 인플루언서 1,000~10,000명
    50~200만 원 마이크로 인플루언서 10,000~100,000명
    200~1,000만 원 미드티어 인플루언서 100,000~500,000명
    1,000만 원+ 매크로 인플루언서 500,000명+

    5. 수익 구조¶

    5.1 중개 수수료 모델¶

    판매자가 인플루언서에게 지급하는 캠페인 계약금에서 수수료 차감.

    거래 유형 수수료율 비고
    기본 캠페인 중개 계약금의 15% 플랫폼이 계약·정산·성과 측정 제공
    어필리에이트 캠페인 매출 발생 시 매출의 5% 선불 없이 성과 기반 정산
    긴급 매칭 (24시간 이내) 계약금의 20% 프리미엄 속성 서비스

    비교 기준 (Shopify Collabs): 판매 발생 시 2.9% 처리 수수료. 셀메이트CM은 더 높은 수수료 대신 더 높은 서비스 수준(매칭·보증·성과 측정) 제공.

    5.2 구독 기능으로 운영 방안¶

    인플루언서 기능을 플랜별로 차등 제공하여 구독 플랜 업셀링에 활용:

    플랜 인플루언서 기능 포함 내용
    무료 인플루언서 DB 탐색만 가능 (프로필 기본 정보)
    스탠다드 월 3건 캠페인 생성, 기본 매칭 추천
    프로 무제한 캠페인, 고급 매칭 알고리즘, ROI 리포트, 어필리에이트 추적
    엔터프라이즈 전담 매니저, 커스텀 인플루언서 풀 구성

    6. 초기 DB 구축 전략 (콜드스타트 해결)¶

    인플루언서 DB가 없으면 매칭 기능 자체가 의미 없다. 콜드스타트를 해결하기 위한 4가지 접근:

    6.1 제3자 데이터 라이선스 계약¶

    • Modash API 또는 유사 플랫폼과 데이터 라이선스 계약 체결
    • 초기 한국·일본 패션·뷰티 카테고리 인플루언서 최소 10만 명 DB 확보
    • 비용: Modash 기준 월 $99~$299 (Starter~Pro 플랜) → 초기 투자 가능 수준

    6.2 인플루언서 자발적 등록 유도 (공급 측 인센티브)¶

    • 얼리버드 인플루언서 프로그램: 먼저 등록한 인플루언서에게 플랫폼 런칭 후 선호 노출 보장
    • 무료 분석 리포트 제공: 자신의 계정을 인증하면 상세 분석 리포트 즉시 제공 (자기 데이터 확인 수단)
    • 협업 수수료 할인: 얼리버드 인플루언서는 초기 캠페인 수수료 10% 감면

    6.3 판매자 현재 파트너 인플루언서 임포트¶

    • 판매자가 온보딩 시 "현재 협업 중인 인플루언서"를 초대할 수 있는 기능 제공
    • 초대받은 인플루언서 → 플랫폼 가입 → DB에 추가
    • 기존 관계를 플랫폼으로 이전하는 자연스러운 온보딩 경로

    6.4 크롤링 보완 (공개 정보 범위 내)¶

    • YouTube 공개 채널 데이터: YouTube Data API v3 (OAuth 불필요)
    • TikTok 공개 영상 메타데이터: TikTok Research API (학술·리서치 목적 신청)
    • Instagram: 공개 API 제약으로 인해 직접 크롤링 불가 → 제3자 데이터 업체 의존
