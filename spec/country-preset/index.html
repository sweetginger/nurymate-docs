    1. 개요 및 문제 정의¶

    1.1 배경¶

    이커머스 판매자가 새로운 국가로 판매를 확장할 때 다음과 같은 설정 영역에서 오류·누락이 빈번하게 발생한다.

    설정 영역 현재 문제
    결제수단 해당 국가 구매자가 선호하는 결제 수단 미지원 → 구매 이탈
    세금 설정 부가세/소비세 요율 미설정 → 세금 미징수 또는 이중 부과
    환불 정책 국가별 법정 청약철회 기간 미준수 → 법적 분쟁 위험
    법적 고지문 표시의무 항목 누락 → 행정처분·과태료 위험
    배송 설정 해당 국가 배송사·배송비 미설정 → 주문 접수 후 배송 불가

    1.2 문제 정의¶

    판매자가 새로운 국가로 판매를 시작할 때 국가별 법적 요건과 비즈니스 관행을 개별적으로 파악하여 설정해야 한다. 이 과정에서:

    • 누락: 법적 필수 고지문을 모르거나 빠뜨림
    • 오류: 세금 요율 잘못 입력, 환불 기간 법정 기준 미충족
    • 지연: 설정 방법 파악에 시간 소요 → 판매 시작 지연
    • 비용: 법률 전문가 자문 비용 발생

    1.3 솔루션 개요¶

    "판매 국가 선택 → 프리셋 자동 적용" 구조를 통해 판매자가 법적·비즈니스적 요건을 일일이 조사하지 않고도 해당 국가에서 즉시 판매 가능한 상태를 만든다.

    벤치마크 — Shopify Markets: Shopify는 국가별 시장(Market) 개념을 도입하여 언어·통화·도메인·세금·가격 설정을 국가 단위로 분리 관리하고, 동적 세금 포함 가격 표시(Dynamic tax-inclusive pricing)를 지원한다. 누리메이트는 여기에 더해 법적 고지문 자동 생성 및 로컬 결제수단 네이티브 통합을 차별점으로 추가한다.

    2. 국가별 프리셋 포함 항목 정의¶

    한국 프리셋¶

    2.1.1 결제수단¶

    결제수단 기본 활성화 여부 비고
    국내 신용카드 (BC·KB·신한·현대·롯데·하나·우리) 활성화 PG 계약 완료 후 자동 활성화
    카카오페이 활성화 월간 이용자 3,800만+
    네이버페이 활성화 네이버 쇼핑 연동 포함
    토스페이 활성화
    무통장 입금 (가상계좌) 활성화 B2B 거래·소규모 판매자 필수
    휴대폰 소액결제 선택 활성화 디지털 콘텐츠 판매 시 권장
    계좌이체 선택 활성화

    2.1.2 배송 정책 템플릿¶

    [기본 배송 정책 — 한국]
- 배송비: 3,000원 (50,000원 이상 무료)
- 배송 기간: 결제 완료 후 1~3 영업일 내 발송
- 배송사: CJ대한통운 / 한진택배 / 롯데택배 (선택)
- 도서산간 지역 추가 배송비: 3,000원~5,000원
- 제주도 추가 배송비: 3,000원

    2.1.3 환불 정책 템플릿 (전자상거래법 기준)¶

    「전자상거래 등에서의 소비자보호에 관한 법률」 제17조 기준:

    [기본 환불 정책 — 한국]
- 청약철회 기간: 상품 수령일로부터 7일 이내 (이유 불문)
- 단순 변심 반품: 왕복 배송비 구매자 부담
- 상품 하자·오배송: 배송비 판매자 부담
- 환불 처리 기간: 반품 수령 후 3 영업일 이내
- 청약철회 불가 사유:
  - 사용 또는 일부 소비로 가치 감소된 경우
  - 포장 훼손으로 상품 가치 현저히 감소된 경우
  - 시간 경과로 재판매 어려운 경우
  - 복제 가능한 디지털 상품 포장 훼손 시

    2.1.4 세금 설정¶

    항목 설정값
    부가가치세 (VAT) 10%
    과세 기준 공급가액
    영수증 발행 전자세금계산서 지원
    간이과세자 여부 판매자 선택 (해당 시 VAT 면제 또는 경감)
    과세 제외 품목 농·수·축산물 등 (판매자 직접 지정)

    2.1.5 필수 법적 고지문 — 한국¶

    통신판매업자 필수 표시사항 (전자상거래법 제13조, 시행령 제21조):

    [쇼핑몰 하단 필수 표시 항목]
1. 상호 및 대표자 성명
2. 사업장 주소
3. 전화번호 및 이메일 주소
4. 사업자등록번호
5. 통신판매업 신고번호
6. 호스팅 서비스 제공자 상호
7. 개인정보 처리방침 링크
8. 이용약관 링크
9. 구매안전서비스(에스크로) 이용 확인 버튼 또는 결제대금 예치 기관명

    일본 프리셋¶

    2.2.1 결제수단¶

    결제수단 기본 활성화 여부 비고
    신용카드 (Visa·MasterCard·JCB·AMEX·Diners) 활성화 JCB 필수 (일본 시장 점유율 높음)
    콘비니 결제 (편의점 결제) 활성화 세븐일레븐·로손·패밀리마트·미니스톱
    은행 자동이체 (銀行振込) 활성화 기업 구매·고액 결제 시 선호
    PayPay 활성화 일본 최대 QR 결제 (이용자 6,000만+)
    LINE Pay 활성화 LINE 앱 기반
    후불 결제 (Paidy · あと払い) 선택 활성화 신용카드 미보유 구매자 대응
    d払い (NTT도코모) 선택 활성화
    au PAY 선택 활성화

    2.2.2 배송 정책 템플릿¶

    [기본 배송 정책 — 일본]
- 배송비: 全国一律 880円 (税込) / 10,000円以上 送料無料
- 배송 기간: ご注文確認後 2～5営業日以内に発送
- 配送業者: ヤマト運輸 / 佐川急便 / 日本郵便 (판매자 선택)
- 沖縄・離島: 追加送料 1,100円
- 時間帯指定: 午前中 / 14-16時 / 16-18時 / 18-20時 / 19-21時
- 置き配: 対応

    주의: 배송 정책은 일본어로 기재되어야 한다. 누리메이트는 위 템플릿을 일본어로 제공하며, 판매자는 한국어 원문과 일본어 표시 문구를 병렬 편집할 수 있다.

    2.2.3 환불 정책 (특定商取引に関する法律 — 통신판매 조항)¶

    일본 특정상거래법상 통신판매는 청약철회(クーリング・オフ) 의무 적용 대상이 아니나, 반품 특약을 명시해야 한다.

    [기본 환불 정책 — 일본]
- 반품 기간: 상품 수령일로부터 8일 이내 (미개봉·미사용)
- 반품 배송비: 상품 불량·오배송의 경우 판매자 부담
- 단순 변심 반품: 구매자 부담 (별도 고지)
- 반품 특약 없을 경우: 법정 8일 이내 반품 허용 의무 발생
- 교환: 재고 있는 경우에 한해 대응
- 환불 처리: 반품 확인 후 7영업일 이내

    [領収書 발행 항목]

- 금액: 税込合計金額

    4.2 프리셋 미리보기 상세¶

    • 미리보기 화면에서 각 항목을 클릭하면 해당 설정의 상세 내용 표시
    • 자동 적용 항목은 적용 전에 변경 불가 (적용 후 개별 수정 가능)
    • 판매자 확인 필요 항목은 미리보기 단계에서 직접 입력 가능
    • 전체 항목 수·자동 적용 수·확인 필요 수를 상단에 요약 표시

    4.3 재적용 및 초기화¶

    • 프리셋 적용 후 개별 설정을 수정한 경우, "프리셋 기본값으로 초기화" 버튼 제공
    • 프리셋 재적용 시 현재 커스터마이즈 내용이 덮어쓰여짐을 경고 표시

    5. 커스터마이즈 가능 범위¶

    5.1 커스터마이즈 원칙¶

    프리셋은 합리적 기본값을 제공하되, 판매자의 비즈니스 상황에 따라 유연하게 조정 가능해야 한다. 단, 법적 최소 요건 이하로의 설정은 차단한다.

    5.2 커스터마이즈 가능 항목¶

    항목 커스터마이즈 가능 범위 제한 사항
    배송비 금액·무료 기준·지역별 추가비 자유 설정 없음
    배송 기간 일수 조정 가능 없음
    반품 기간 8일 이상으로만 설정 가능 (일본 기준) 법정 최소 8일 미만 불가
    청약철회 기간 7일 이상으로만 설정 가능 (한국 기준) 법정 최소 7일 미만 불가
    세금 (VAT/소비세) 경감세율 품목 지정 가능 기본 세율(10%) 임의 변경 불가
    결제수단 계약된 결제수단 on/off 계약 미완료 수단은 활성화 불가
    법적 고지문 추가 문구 입력 가능 필수 항목 삭제 불가
    영수증 양식 但し書き(용도) 기본값 설정 가능 필수 기재 항목 삭제 불가

    5.3 법적 요건 보호 로직¶

    • 법정 최소 요건 이하 설정 시도 시 인터랙티브 경고 표시
    • 경고 메시지 예시: "일본 특정상거래법상 반품 특약 미기재 시 법정 8일 이내 반품을 반드시 허용해야 합니다. 최소 8일 이상으로 설정해주세요."
    • 경고 무시 후 저장 차단 (법적 보호 항목은 저장 자체를 막음)

    6. 지원 국가 확장 로드맵¶

    Phase 1 — 한국 (MVP, 2026 Q2)¶

    • 대상: 한국 내 판매 (기본값)
    • 우선순위: 모든 판매자의 기본 출발점
    • 포함 항목: 한국 프리셋 전체 (§2.1)
    • 목표: 통신판매업 신고 정보 연동, 부가세 신고 자료 자동 생성

    Phase 2 — 일본 (2026 Q3~Q4)¶

    • 대상: 한국 판매자의 일본 크로스보더 판매
    • 포함 항목: 일본 프리셋 전체 (§2.2)
    • 추가 필요 작업:
    • 일본어 법적 고지문 번역 검수 (법무 전문가)
    • 일본 결제 대행사 계약 연동 (콘비니·PayPay 등)
    • 인보이스 제도 연동 (适格請求書발행사업자 등록번호 입력)

    Phase 3 — 동남아시아 (2027 Q1~Q2)¶

    국가 우선 지원 이유 주요 결제수단
    태국 한류 소비 강세, K-뷰티 수요 PromptPay, 편의점 결제
    싱가포르 영어권, 규제 투명 PayNow, 신용카드
    말레이시아 이슬람 할랄 인증 필요성 DuitNow, FPX
    인도네시아 동남아 최대 시장 QRIS, GoPay, OVO
    • 주요 추가 요건: 현지 소비자보호법, 데이터 현지화 규정, 환율 정책

    Phase 4 — 글로벌 (2027 Q3~)¶

    • 미국: CCPA(캘리포니아 소비자 프라이버시법), 주별 세금 Nexus 설정
    • EU: GDPR 고지문, VAT OSS(One Stop Shop) 신고
    • 호주: GST 10%, Australian Consumer Law 14일 반품
    • 영국: UK GDPR, VAT 20%

    • 법률 변경 후 업데이트 알림 발송 시점부터 미적용에 따른 법적 책임은 판매자에게 귀속됨을 고지

    8.2 MVP 정의 (Phase 1)¶

    Phase 1 MVP는 다음 조건을 충족해야 한국 판매 시작이 법적으로 안전하다고 판단할 수 있다:

    1. 부가세 10% 자동 적용
    2. 전자상거래법 기준 청약철회 7일 정책 템플릿 제공
    3. 필수 하단 표시 항목 자동 생성 (사업자 정보 기반)
    4. 개인정보처리방침·이용약관 링크 자동 삽입
    5. 카카오페이·네이버페이·토스페이 기본 활성화 (PG 계약 완료 전제)
