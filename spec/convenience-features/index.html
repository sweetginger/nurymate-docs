    1. 개요¶

    1.1 배경¶

    셀메이트CM은 셀러가 상품을 판매하는 것을 넘어, 운영 자동화를 통해 실질적인 시간을 돌려주는 플랫폼을 목표로 한다. 셀러가 야간에 프로모션 시작을 위해 수동으로 게시하거나, 재고 소진 여부를 직접 확인해야 하는 상황은 SaaS 플랫폼의 경쟁력을 약화시킨다.

    국내 1위 카페24는 운영 자동화 기능을 네이티브로 광범위하게 제공하고 있다. 셀메이트CM이 카페24의 대체재로 포지셔닝하려면 이 영역에서 동급 이상의 기능을 제공해야 한다.

    1.2 목표¶

    • 셀러가 스케줄링 기능만으로 야간·주말 프로모션을 자동 운영할 수 있는 환경 제공
    • 반복적인 수동 작업(가격 변경, 재고 확인, 게시/숨김 전환)을 자동화
    • 카페24 대비 더 직관적인 UI로 동급 이상 기능 제공

    1.3 범위¶

    범주 설명
    스케줄링 상품·배너·쿠폰 게시·종료 시각 예약
    재고 자동화 품절 처리, 자동 숨김, 재입고 알림
    가격 자동화 세일 시작·종료 자동 처리
    운영자 알림 재고 경고, 신규 주문, 정산 완료 등
    일괄 관리 가격·카테고리·노출 순서 벌크 처리

    2. 타사 현황 벤치마크¶

    2.1 상품 게시·숨김 예약¶

    플랫폼 지원 방식 상세
    카페24 ✅ 네이티브 (강력) 진열 예약: 날짜·시각(분 단위) 지정 자동 진열/해제. 배치 예약(CSV 업로드). 예약 캘린더 뷰. 자동 진열 30개 제한
    식스샵 ✅ 판매 스케줄러 판매 기간(시작·종료일) 설정으로 자동 판매 시작·종료. 게시상태(진열/숨김)와 판매상태(판매중/판매안함) 조정은 사실상 같은 역할 → 결과적으로 동일 효과 달성. 진열 순서 예약 미지원
    아임웹 ✅ 판매기간 설정 상품별 판매 시작일·종료일 분 단위 설정. 게시상태(진열/숨김)와 판매상태(판매중/판매안함) 조정은 사실상 같은 역할 → 결과적으로 동일 효과 달성. 진열 순서 예약 미지원
    Shopify ⚠️ 제한적 + Flow Online Store 채널 한정 미래 게시 가능. Unpublish 예약 불가. Shopify Flow로 재고 기반 자동화 가능. 완전 자동화는 앱 필요
    BigCommerce ⚠️ 제한적 Product Availability 수동 제어. 전용 "게시 예약" UI 없음. 서드파티 앱·API 활용 필요
    WooCommerce ⚠️ WP 예약 기능 WordPress 게시물 예약으로 상품 게시 가능. Unpublish 예약은 플러그인 필요

    시사점: 게시상태(진열/숨김)와 판매상태(판매중/판매안함) 조정은 사실상 같은 역할을 한다 — 어느 쪽을 제어하든 결과적으로 "특정 시점에 상품이 고객에게 노출·구매 가능한지"를 결정한다. 식스샵·아임웹이 "독립적인 게시 예약 UI"를 별도로 제공하지 않는 것은 한계가 아니라, 판매 기간 설정을 통해 동일한 결과를 다른 방식으로 달성하는 것이다. 카페24만의 실질적 차별점은 배치 예약(CSV 업로드)과 예약 캘린더 뷰, 그리고 진열 순서 예약이며, Shopify·BigCommerce는 채널 제한 또는 앱 의존이 여전히 한계.

    2.2 배너·팝업 노출 기간 예약¶

    플랫폼 지원 방식 상세
    카페24 ✅ 네이티브 (강력) 팝업별 노출 기간(시작~종료) 설정. 상태 3단계(대기/진행/종료). 특정 쿠폰 보유 회원 타겟팅. PC/모바일 별도 설정
    아임웹 ✅ 네이티브 (중간) 팝업 노출 기간 설정 지원. 데스크톱/모바일 선택. 전체/회원 전용 구분. 고급 타겟팅(구매이력 기반) 제한
    식스샵 ⚠️ 외부 앱 (식스샵 프로) 식스샵 프로에서 코드앤버터 제휴로 스마트 팝업·배너 제공. 구매 가능성 높은 시점 개인화 노출. 독자적 노출 기간 스케줄링 UI 없음
    Shopify ❌ 앱 필요 네이티브 팝업 기능 없음. Privy, OptiMonk 등 앱에서 노출 기간·타겟팅 설정
    BigCommerce ❌ 앱 필요 배너 관리 기능 있으나 스케줄링 제한. 전문 팝업·배너 앱 필요
    WooCommerce ❌ 플러그인 필요 Popup Maker, Elementor Pro Popup 등으로 구현

    시사점: 카페24·아임웹은 팝업 노출 기간을 네이티브 지원. 식스샵·Shopify·BigCommerce는 이 영역에서 외부 앱 의존 → 셀메이트CM은 네이티브 지원으로 차별화 가능.

    2.3 쿠폰·프로모션 자동 시작·종료¶

    플랫폼 지원 방식 상세
    카페24 ✅ 네이티브 (강력) 쿠폰 사용 기간(시작~종료) 설정. 쿠폰 정기 자동 발급. 기간 할인 등록. 발급 일시중지·재개 설정
    식스샵 ✅ 기본 지원 쿠폰 사용 기간(시작~종료일) 설정. 가입·구매완료·후기 작성·추천인 조건별 쿠폰 자동지급. 할인 코드(제휴 마케팅용) 지원. 쿠폰 발급 시 카카오 알림톡/SMS/이메일 자동 발송
    아임웹 ✅ 기본 지원 쿠폰 생성·배포, 사용 기간 설정 지원. 쿠폰과 예약 기능 비통합(예약 상품에 쿠폰 사용 불가)
    Shopify ✅ 기본 + 앱 확장 자동 할인(Automatic Discounts) 기본 지원. 쿠폰 만료일 설정 가능. 할인 시작일 즉시만 가능. Flow로 쿠폰 발급 자동화
    BigCommerce ✅ 네이티브 (강력) 70가지 이상 네이티브 할인 옵션. 프로모션 시작·종료일 설정. 대량 쿠폰 코드 생성. 앱 없이 복잡한 프로모션 관리 가능
    WooCommerce ✅ 기본 지원 쿠폰 만료일 설정 네이티브 지원. 고급 스케줄링은 플러그인

    시사점: 쿠폰 자동 시작·종료는 대부분 플랫폼이 기본 지원. BigCommerce가 70가지 이상 옵션으로 가장 강력. 식스샵·카페24는 카카오 알림톡 연동이 한국 생태계 강점.

    2.4 재고 소진 자동 품절·숨김 처리¶

    플랫폼 품절 처리 자동 숨김 재입고 고객 알림
    카페24 ✅ 네이티브 ✅ 네이티브 ✅ 카카오 알림톡/SMS
    식스샵 ✅ 네이티브 ❌ 수동 ❌ 미지원 (알림톡 발송은 별도 설정)
    아임웹 ✅ 네이티브 ❌ 수동 ❌ 제한적
    Shopify ✅ 네이티브 ❌ Flow/앱 필요 ❌ 앱 필요
    BigCommerce ✅ 네이티브 ✅ 네이티브 ❌ 앱 필요
    WooCommerce ✅ 네이티브 ❌ 플러그인 ❌ 플러그인

    상세 사항: - 카페24: 재고 0 시 자동 품절 표시 + 주문 불가. 품절 상품 자동 후순위 진열. 재입고 시 카카오 알림톡/SMS 자동 발송 - BigCommerce: 재고 소진 시 카테고리·검색 결과에서 숨김 또는 품절 표시 선택 가능 - Shopify: Shopify Flow로 재고 0 → 자동 비공개 워크플로우 구성 가능하지만 추가 설정 필요

    시사점: 재입고 고객 알림 자동화는 카페24만 네이티브로 지원. 셀메이트CM은 카카오 알림톡·이메일 연동 재입고 알림으로 확실한 차별화 포인트.

    2.5 가격 예약 변경 (세일 자동화)¶

    플랫폼 지원 방식 상세
    카페24 ⚠️ 프로모션으로 우회 기간 할인 혜택 등록으로 세일 기간 자동 시작·종료. 직접적인 가격 예약 변경 UI는 부재
    식스샵 ⚠️ 판매 스케줄러로 우회 판매 기간(시작·종료) 설정으로 상품 자동 판매·중단. 세일가 → 정가 자동 복원 기능 없음
    아임웹 ⚠️ 판매기간으로 우회 판매 기간 설정으로 타임 세일 구현. 판매 종료 = 판매 중단 (가격 자동 복원 아님)
    Shopify ❌ 앱 필요 Compare-at price 표시 지원. 가격 예약 변경 자체는 네이티브 미지원. BOLD Discounts, Easy:Sale 등 앱 필수
    BigCommerce ✅ 네이티브 (강력) Sale Price 설정 + 프로모션 시작·종료일로 자동 가격 변경. Automatic Promotions으로 장바구니 레벨 자동 할인
    WooCommerce ✅ 기본 지원 Regular Price + Sale Price 이중 설정. Sale Price에 시작·종료 날짜 지정 가능 (네이티브)

    시사점: 직접적인 가격 예약 변경(정가 → 세일가 → 정가 자동 복원)은 BigCommerce·WooCommerce가 네이티브 지원. 카페24·식스샵·Shopify는 프로모션·앱·스케줄러로 우회. 셀메이트CM은 이 영역을 단순한 UI로 커버하면 차별화 가능.

    시사점: 알림 채널 다양성(이메일·SMS·카카오·슬랙)과 이벤트 세분화가 핵심 경쟁 요소.

    3. 셀메이트CM 반영 방향¶

    3.1 통합 스케줄링 캘린더 (핵심 차별화)¶

    카페24는 각 기능별로 스케줄링 설정이 분산되어 있다. 셀메이트CM은 하나의 캘린더 뷰에서 상품·배너·쿠폰·가격 변경 예약을 통합 관리하는 UI를 제공한다.

    운영자 워크플로우: 

    캘린더 뷰 → 날짜 선택 → 이벤트 추가 (상품/배너/쿠폰/가격)
→ 시작·종료 시각 지정 → 예약 완료 → 자동 실행

    상품 게시 예약: - 날짜·시각(분 단위) 지정 후 자동 게시/숨김 - 배치 예약: CSV 업로드 또는 목록 다중 선택 - 예약 상태 배지: 관리자 목록에 "D-3 예약" 형태로 표시

    배너·팝업 예약: - 메인 슬라이더·팝업·공지 배너 노출 시각 예약 - 다중 배너 스케줄: 기간별 우선순위 번호로 순서 결정 - 예약 기간 종료 후 이전 배너 자동 복원

    3.2 가격 예약 변경 (단순화 핵심)¶

    카페24·아임웹보다 직관적인 가격 스케줄링 UI 제공:

    상품 편집 → 가격 섹션
├── 정가: 30,000원
├── 세일가: 22,000원 (26% 할인)
└── 세일 기간: 2026-03-25 09:00 ~ 2026-03-28 23:59
    → 세일 시작 시 자동 세일가 적용
    → 세일 종료 시 자동 정가 복원

    배치 가격 변경: 카테고리 내 전체 상품에 동일 할인율 일괄 적용 예약 가능.

    3.3 재고 자동화¶

    기능 설정 동작
    자동 품절 처리 재고 추적 활성화 시 기본 동작 재고 0 → 주문 불가, 품절 표시
    자동 후순위 진열 옵션 (기본 ON) 품절 상품 → 카테고리 최하단 이동
    자동 숨김 옵션 (기본 OFF) 품절 상품 → 스토어 비노출
    재입고 알림 고객 신청 기반 재입고 시 이메일/카카오/SMS 자동 발송
    저재고 운영자 알림 임계값 설정 재고 N개 이하 시 운영자 알림 채널 발송

    예약 주문 모드: 재고 0 상태에서 "예약 주문" 버튼 활성화, 재입고 예정일 표시 옵션.

    3.4 운영자 알림 체계¶

    알림 이벤트 목록: - 신규 주문 접수 - 주문 결제 완료 - 재고 임계값 도달 (임계값 직접 설정) - 품절 상품 재입고 (운영자) - 정산 완료 / 정산 예정 - 배송 완료 (환불 요청 기간 개시) - 반품·환불 접수

    알림 채널 (우선순위 순): 1. 이메일 (기본, 무료) 2. 카카오 알림톡 (유료, 한국 운영자 핵심) 3. 웹훅 (슬랙, 팀즈 등 외부 채널 자체 연동)

    3.5 벌크 상품 관리¶

    기능 지원 방식
    가격 일괄 변경 목록 다중 선택 + 금액/비율 입력
    세일가 배치 설정 카테고리 단위 할인율 일괄 적용
    카테고리 일괄 이동 드래그 또는 목록 선택 후 이동
    노출 순서 관리 드래그앤드롭 + CSV 업로드
    진열 예약 날짜·시각 지정 자동 순서 변경
    상태 일괄 변경 판매중/품절/숨김 일괄 전환

    4. 구현 우선순위 매트릭스¶

    5. 기술 고려사항¶

    5.1 스케줄링 아키텍처¶

    [운영자 어드민] → 예약 설정 저장
       ↓
[Schedule Store] (DB: scheduled_events 테이블)
       ↓
[Job Scheduler] (Celery Beat / AWS EventBridge)
       ↓
[실행 시각 도달] → 상품 상태 변경 / 가격 변경 / 배너 활성화
       ↓
[CDN 캐시 무효화] → 변경사항 즉시 반영

    핵심 요건: - 분(minute) 단위 정확도 보장 - 예약 실패 시 재시도 + 운영자 알림 - 다중 예약 충돌 감지 (동일 상품에 겹치는 예약)

    5.2 재고 이벤트 처리¶

    주문 완료 이벤트 발생
       ↓
재고 차감 (write DB)
       ↓
재고 임계값 체크 → 저재고 알림 발송
       ↓
재고 = 0 → 품절 처리 이벤트 발행
       ↓
자동 후순위 진열 / 자동 숨김 (설정에 따라)
       ↓
재입고 알림 대기자 목록 활성화

    5.3 알림 채널 추상화¶

    외부 알림 채널을 추상화하여 채널 추가 시 기존 코드 수정 없이 확장 가능한 구조:

    class NotificationChannel(ABC):
    async def send(self, recipient, event_type, payload): ...

class EmailChannel(NotificationChannel): ...
class KakaoChannel(NotificationChannel): ...
class WebhookChannel(NotificationChannel): ...

    5.4 헤드리스 호환성¶

    셀메이트CM은 헤드리스 커머스 구조이므로 운영 편의기능이 프론트엔드 구현에 독립적이어야 한다: - 모든 상태 변경(게시·품절·가격)은 API 이벤트로 프론트에 전파 - 스토어프론트 캐시 무효화 웹훅 제공 - 재입고 알림 신청·조회 API 제공 (프론트가 직접 구현 가능)

    식스샵 운영 편의기능 요약¶

    식스샵은 2019년 런칭한 국내 SaaS 쇼핑몰 플랫폼으로, 카페24·아임웹 대비 진입 장벽이 낮고 중소형 셀러를 주 타겟으로 한다. 2023년 출시한 식스샵 프로에서 운영 편의기능을 강화했다.

    운영 자동화 지원 현황: - 판매 스케줄러: 상품별 판매 시작·종료일 설정으로 자동 판매 시작·중단 (카페24 진열 예약 대비 기능 제한적) - 주문 처리: 부분 배송·부분 반품 처리 지원 (단건 주문 내 상품별 처리) - 재고 관리: 상품 코드·재고 수량 관리, 재고/카테고리/배송 방식별 분류. 재고 소진 품절 표시 기본 지원. 자동 숨김 및 재입고 알림은 미지원 - 통계: 식스샵 프로에 통계 메뉴 추가 (매출·방문자 기본 통계) - 멀티채널 관리: 스마트스토어·쿠팡·무신사 등 외부 채널 주문을 한 곳에서 관리 - 운영자 알림: 주문 접수·배송 단계별 카카오 알림톡/SMS/이메일 자동 발송

    카페24 대비 주요 한계: - 진열 예약·예약 캘린더 없음 - 재고 자동 숨김·재입고 고객 알림 미지원 - 가격 예약 변경(세일 자동화) 없음 - 배너·팝업 노출 기간 자체 스케줄링 없음 (외부 앱 의존)
