    배송 연동 기능 기획서¶

    문서 번호: SPEC-SHIP-001 작성일: 2026-03-11 최종 수정: 2026-03-12 상태: Draft 관련 태스크: [12-6], [14-1] 연관 문서: research/logistics-strategy.md, spec/sellex-integration.md

    1. 개요¶

    1.1 목표¶

    판매자가 누리메이트 어드민에서 운송장을 출력하고, 배송 상태를 실시간 추적하며, 구매자가 자신의 배송 현황을 한 곳에서 확인할 수 있는 통합 배송 관리 기능.

    1.2 설계 원칙¶

    • 모듈 기반 연동: 택배사·3PL과 직접 계약하지 않고, 셀메이트 OMS·WMS 모듈과 셀익스 해외배송모듈을 경유하는 구조
    • 다택배사 단일 뷰: 여러 택배사를 이용하더라도 판매자·구매자 모두 하나의 화면에서 조회
    • 판매자 자립 운영: 별도 물류 시스템 없이 어드민에서 바로 처리 가능
    • 3PL 확장성: 셀메이트 제휴 3PL 리스트를 누리메이트 어드민에서 선택·연동 가능

    2. 배송 연동 구조¶

    누리메이트는 택배사·3PL사와 직접 API를 개발하지 않는다. 셀메이트와 셀익스 모듈을 경유하여 배송 기능을 제공한다.

    2.1 국내배송 — 셀메이트 OMS 택배통합모듈¶

    셀메이트 OMS(Order Management System) 택배통합모듈을 통해 한국 국내 택배사와 연동한다.

    [누리메이트 어드민]
    │
    ▼
[셀메이트 OMS 택배통합모듈 API]
    │
    ├─ CJ대한통운  (국내 1위, 시장점유율 약 50%)
    ├─ 한진택배    (국내 2위)
    ├─ 롯데택배    (국내 3위)
    ├─ 우체국택배  (지방·도서 지역 커버리지 강점)
    └─ 로젠택배    (중소 판매자 이용 많음)

    연동 내용: - 운송장 발급 신청 → 셀메이트 택배 통합 모듈 경유 - 배송 조회 API → 셀메이트 택배 통합 모듈 경유 후 누리메이트 화면에 표시 - 누리메이트 판매자는 어드민에서 택배사를 선택하면 나머지는 자동 처리됨

    2.2 해외배송 — 셀익스 해외배송모듈¶

    셀익스(SellEX) 해외배송모듈을 통해 해외배송을 연동한다.

    [누리메이트 어드민]
    │
    ▼
[셀익스 해외배송모듈 API]
    │
    ├─ 일본 (사가와, JP POST 등)
    ├─ 동남아 (DHL, FedEx, EMS 등)
    └─ 기타 국제 배송사

    연동 내용: - 해외 배송국가 선택 → 셀익스 모듈이 최적 배송사 매칭 - 국제 운송장 발급 및 추적 → 셀익스 모듈 경유 - 관세·통관 서류 자동화 → 셀익스 기능 활용

    2.3 3PL 연동 — 셀메이트 WMS 모듈 + 제휴 3PL 리스트¶

    셀메이트 제휴 3PL 업체 리스트를 누리메이트 어드민에서 조회·선택할 수 있다. 3PL 연동 시 셀메이트 WMS(Warehouse Management System) 모듈이 재고 관리를 담당한다.

    [누리메이트 어드민] — 3PL 선택 UI
    │
    ▼
[셀메이트 WMS 모듈] — 재고 관리
    │
    └─ + 셀메이트 제휴 3PL 추가 목록

    연동 내용: - 판매자가 어드민에서 3PL 업체 선택 → 셀메이트 WMS API 연동 활성화 - 주문 발생 시 자동으로 3PL 창고에 출고 요청 전달 - 3PL 출고 후 운송장 번호 자동 등록 → 배송 추적 연동

    3. 핵심 기능 정의¶

    3.1 배송비 정책 설정¶

    정책 유형 설명
    기본 배송비 주문당 고정 금액 (예: 3,000원)
    무료배송 조건 특정 금액 이상 무료 (예: 50,000원 이상 무료)
    지역별 추가 요금 제주·도서산간 추가 배송비 설정
    무게·부피 기준 상품 무게 또는 부피 기준 배송비 계산
    무료배송 상품 개별 설정 특정 상품만 무료배송 지정

    3.2 운송장 발행 및 출력¶

    [판매자 어드민] 주문 관리
    │
    ├─ 주문 선택 → 택배사 선택 → 운송장 번호 입력 (수동)
    │   또는
    └─ 셀메이트 OMS 모듈 → 운송장 자동 발급 신청
         │
         ▼
    운송장 PDF 자동 생성 → 프린터 출력

    일괄 처리 지원: 주문 100건 선택 → 운송장 일괄 발급 → ZIP 다운로드

    3.3 배송 추적 통합 뷰¶

    판매자 어드민¶

    +=========================================================+
|  배송 현황                                              |
+=========================================================+
| 주문번호  | 구매자 | 택배사  | 운송장번호 | 현재 위치   |
+-----------+-------+---------+-----------+-------------+
| #10234   | 김○○  | CJ대한  | 123456789 | 출고 완료   |
| #10235   | 이○○  | 한진    | 987654321 | 배송 중     |
| #10236   | 박○○  | 롯데    | 456789123 | 배달 완료 ✅ |
+=========================================================+
    • 전체 주문의 배송 상태를 단일 화면에서 조회
    • 배송 이상 (분실·오배송·지연) 자동 감지 및 알림
    • 미배달 장기 주문 자동 추출 (5일 이상 미배달 등)

    구매자 화면 (스토어 프론트)¶

    • 주문 조회 페이지에서 실시간 배송 단계 표시 (진행 바 형태)
    • SMS·이메일 배송 알림 자동 발송 (발송, 배달 출발, 배달 완료)

    3.4 반품·교환 배송¶

    기능 내용
    반품 배송비 설정 반품 귀책별 배송비 부담 주체 설정
    반품 운송장 자동 발급 구매자 반품 신청 시 반품 운송장 자동 생성·발송
    교환 배송 처리 교환 상품 발송 시 원클릭 운송장 발급

    4. 3PL 연동 플로우 (Phase 2)¶

    4.1 목표¶

    판매자가 누리메이트 어드민에서 셀메이트 제휴 3PL 업체를 선택하면, 주문이 자동으로 해당 3PL 창고에 전달되어 출고 처리되는 풀필먼트 연동.

    4.2 3PL 연동 구조¶

    [누리메이트 주문 발생]
         │
         ▼
[3PL 연동 활성화 여부 확인]
         │
    ┌────┴────┐
   예         아니오
    │           │
    ▼           ▼
[셀메이트 WMS  [판매자 직접
  모듈 전달]    발송 처리]
         │
         ▼
[3PL 출고 처리 → 운송장 자동 등록]
         │
         ▼
[누리메이트 주문 상태 자동 업데이트]

    4.3 누리메이트 어드민 3PL 선택 UI¶

    • 셀메이트 제휴 3PL 리스트 자동 로드 (셀메이트 파트너 API 연동)
    • 업체별 특징·지역·카테고리 필터 제공 (예: 냉장 가능 여부, 당일출고 지역)
    • 선택 시 연동 활성화 → 셀메이트 WMS 모듈과 자동 페어링

    5. 콘비니 픽업 연동 (Phase 3 — 일본)¶

    일본 소비자가 편의점(세븐일레븐·패밀리마트·로손)에서 상품을 수령하는 콘비니 픽업 기능.

    • 상세 기획: research/japan-conveni-pickup.md 참조
    • 기술 요건: 셀익스 해외배송모듈 내 콘비니 픽업 지원 여부 확인 후 연동 방식 결정

    6. 구현 우선순위¶

    Phase 기능 일정
    P0 (MVP) 운송장 수동 입력 + 셀메이트 OMS 모듈 연동 (국내 택배사 조회) 2026 Q2
    P0 배송비 정책 설정 (기본·무료조건·지역 추가) 2026 Q2
    P0 구매자 배송 추적 페이지 2026 Q2
    P1 셀메이트 OMS 모듈 경유 운송장 자동 발급 2026 Q3
    P1 배송 이상 감지 알림 2026 Q3
    P1 셀익스 해외배송모듈 연동 2026 Q3
    P2 셀메이트 WMS 모듈 + 3PL 제휴 리스트 연동 2026 Q4
    P3 콘비니 픽업 연동 (셀익스 모듈 경유) 2027 H1

    7. 성공 지표 (KPI)¶

    지표 목표값
    배송 추적 갱신 주기 ≤ 2시간 (배치)
    배송 알림 발송 성공률 ≥ 99%
    운송장 자동 발급 성공률 ≥ 99.5%
    배송 관련 CS 문의 비율 ≤ 5% (전체 주문 대비)
