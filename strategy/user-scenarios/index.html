          • S7-02. 일본 결제 패키지 설정 (콘비니·코드결제·카드 자동 활성화)

        • 카테고리 8. 번역 관리

    S1-01. 30분 AI Setup Wizard — 가입부터 라이브까지¶

    셀러는 카카오로 로그인하고 판매 카테고리와 국가(한국)를 선택한다. AI 어시스턴트가 5단계 Setup Wizard(기본정보 → 테마 → 상품 → 결제 → 오픈 준비)를 진행하면서 단계마다 맥락에 맞는 팁을 제공한다. 우측 패널에서는 입력한 내용이 실제 스토어에 어떻게 보이는지 실시간으로 미리 확인할 수 있다. 이탈해도 진행 상태가 저장되어 다시 접속하면 멈춘 단계부터 이어서 진행할 수 있다. 30분 후 스토어 URL이 생성되고 결제 가능한 상태가 된다.

    S1-02. AI 상품 설명 자동 생성 — 사진 한 장, 2분 완성¶

    셀러가 스마트폰으로 찍은 상품 사진을 드래그해서 올린다. AI가 이미지를 분석해 상품명·상세설명·카테고리·추천 태그를 자동으로 생성하고 편집 가능한 형태로 보여준다. 셀러는 가격과 재고 수량만 입력하고 등록 버튼을 누른다. 총 소요 시간 2분.

    S1-03. 카페24 마이그레이션 — 개발자 없이 6단계 마법사¶

    카페24를 5년 쓴 셀러가 마이그레이션 마법사를 시작한다. 카페24 OAuth 인증을 완료하자 사전 분석이 자동으로 실행되어 상품 1,247개·이미지 3,891장·예상 소요 시간 2시간 45분이 화면에 표시된다. 실행 버튼을 누르면 브라우저를 닫아도 백그라운드에서 계속 처리되며, 완료 시 카카오 알림이 온다. 이전이 끝나면 정합성 검증이 자동 실행되고 "상품 1,234/1,247 완료, 오류 13건" 리포트와 오류 목록 CSV가 제공된다.

    S1-04. 기존 도메인 연결 — 기술 지식 없이 20분¶

    셀러는 가비아에서 구매해 둔 도메인을 입력창에 붙여넣는다. 셀메이트CM이 호스팅사별 스크린샷을 포함한 DNS 설정 가이드를 자동으로 보여주고, DNS 전파 여부를 자동으로 감지해 상태 배지를 실시간으로 업데이트한다. 설정 완료가 확인되는 순간 SSL 인증서가 자동 발급되고 커스텀 도메인이 활성화된다.

    S1-05. 셀메이트CM에서 도메인 구매 — 원스톱¶

    따로 구매해 둔 도메인이 없는 셀러가 원하는 이름을 검색한다. .com·.co.kr·.shop 등 사용 가능한 도메인 목록과 가격이 표시되고, Gabia·Namecheap 파트너 연동으로 셀메이트CM 화면을 벗어나지 않고 결제가 완료된다. 구매 직후 DNS 설정과 SSL 발급이 자동 처리되어 별도 도메인 관리 화면에 접속할 필요가 없다.

    S1-06. 법적 고지문 자동 생성 — 통신판매업 필수 표시 완료¶

    셀러가 사업자 등록번호와 상호명을 입력하자 셀메이트CM이 통신판매업 필수 표시사항, 이용약관, 개인정보처리방침을 표준 템플릿으로 자동 생성한다. 한국 전자상거래법 요건에 맞는 내용이 스토어 하단에 자동 삽입된다. 셀러는 법률 지식 없이도 법적 요건을 충족한 스토어를 오픈할 수 있다.

    카테고리 2. 일상 운영¶

    S2-01. AI 채팅으로 배송비 변경 — 메뉴 찾을 필요 없음¶

    셀러가 어드민 하단의 AI 채팅(Ctrl+K)에 "5만 원 이상 무료배송으로 바꿔줘"라고 입력한다. AI가 현재 설정(항상 3,000원)과 변경 후 내용을 나란히 보여주며 "이대로 변경할까요?"를 묻는다. 확인을 누르면 즉시 적용되고, 24시간 이내에는 "되돌리기" 버튼으로 롤백이 가능하다. 카페24에서 배송비 하나 바꾸려면 5단계 메뉴를 거쳐야 했던 것과 대비된다.

    S2-02. 장바구니 이탈 자동 카카오 알림톡¶

    셀러가 마케팅 설정에서 "장바구니 이탈 알림" 토글을 켠다. 이후 구매자가 장바구니에 상품을 담고 1시간 동안 결제하지 않으면 "아직 장바구니에 남아 있어요" 카카오 알림톡이 자동으로 발송된다. 셀러는 별도 작업 없이 이탈 구매자에게 자동으로 재접촉한다.

    S2-03. 리뷰 요청 자동 발송 — 배송 완료 다음 날¶

    셀러는 리뷰 요청 자동화를 켜두고 평소대로 운영한다. 배송 완료 D+1에 구매자에게 "주문하신 상품 잘 받으셨나요? 리뷰를 남겨주시면 포인트를 드려요" 카카오 알림이 자동으로 발송된다. 텍스트 리뷰 50P, 사진 리뷰 200P로 차등 지급되며 셀러는 수동으로 리뷰 요청 메시지를 보낼 필요가 없다.

    S2-04. AI 주간 인사이트 — "왜 이번 주 매출이 올랐지?"¶

    월요일 아침 셀러의 어드민 홈에 AI가 작성한 주간 인사이트가 표시된다. "지난 주 매출이 32% 증가했어요. 금요일 인스타그램 포스팅 이후 트래픽이 집중됐고, 블라우스 상품의 전환율이 평소 대비 1.8배 높았어요." 셀러는 데이터를 직접 분석하지 않아도 무엇이 효과적이었는지 즉시 파악할 수 있다.

    S2-05. AI 채팅으로 시즌 가격 일괄 변경¶

    봄 시즌 상품 가격을 10% 올려야 하는 셀러가 채팅에 "봄 컬렉션 태그 달린 상품 가격 10% 올려줘"라고 입력한다. AI가 해당 태그 상품 67개의 가격 변경 전·후를 미리보기로 보여주며 확인을 요청한다. 승인하면 67개 상품 가격이 한 번에 업데이트되고 변경 이력이 로그에 남는다. 롤백은 24시간 이내 가능하다.

    S2-06. AI 재고 소진 예측 알림 — 품절 사고 사전 차단¶

    셀러가 따로 재고를 일일이 확인하지 않아도, 셀메이트CM AI가 일평균 판매량을 분석해 "현재 재고 기준으로 3일 후 블라우스(M, 화이트)가 소진될 예정이에요"라는 알림을 준다. 셀러는 품절로 인한 주문 취소 없이 적시에 재고를 확보할 수 있다.

    카테고리 3. 결제 / 정산¶

    S3-01. PG 심사 전에도 판매 시작 — 무통장 즉시 활성화¶

    막 개설한 셀러가 카드 결제 PG 심사가 완료되지 않은 상태다. 셀메이트CM은 PG 심사와 무관하게 무통장입금·가상계좌를 즉시 활성화하며, 테스트 결제 시뮬레이션으로 결제 플로우가 정상인지 미리 확인할 수 있다. 셀러는 PG 심사 결과를 기다리는 동안에도 주문을 받기 시작할 수 있다.

    S3-03. 일본 결제 패키지 원클릭 활성화 — Shopify 대비 4주 단축¶

    Shopify를 쓰는 일본 진출 셀러는 KOMOJU 연동에만 4~5주가 걸리고, 수수료도 KOMOJU 3.6% + Shopify 서드파티 2.0% 이중 부담이었다. 셀메이트CM에서는 결제 설정 페이지에서 "일본 결제 패키지" 토글 하나를 켜면 GMO PG 기반 신용카드(JCB·VISA)·PayPay·콘비니 결제가 한 번에 활성화된다. 한국 법인으로도 일본 QR 결제에 접근할 수 있고, 이중 수수료 구조 없이 GMO PG 단일 수수료만 부담한다.

    S3-04. 통합 정산 대시보드 — 한국·일본 채널 한눈에¶

    한국 쇼핑몰과 일본 스토어를 동시에 운영하는 셀러가 이번 달 국가별 매출을 확인하고 싶다. 통합 정산 대시보드에서 채널별·국가별·결제수단별 정산 내역이 하나의 화면에 집계된다. 별도 스프레드시트 없이 원화 기준으로 환산된 전체 매출과 각 채널의 순수익을 한 번에 파악할 수 있다.

    S3-05. 부분 환불 자동 처리 — 3개 중 1개 반품¶

    구매자가 3개 묶음 주문 중 1개만 반품 신청을 한다. 셀러는 주문 상세 화면에서 반품 수량을 '1'로 입력하고 환불 처리를 누른다. 1개에 해당하는 금액이 자동 계산되어 PG사로 환불 요청이 전송되고, 나머지 2개 주문은 정상 완료 상태로 유지된다. 수동 계산 없이 클릭 한 번으로 끝난다.

    S3-06. 해외 구매자 결제 → 원화 정산¶

    해외 쇼핑객이 셀러의 스토어에서 달러로 결제한다. 셀메이트CM은 Payoneer 파트너십 기반으로 크로스보더 결제를 처리하고, 셀러는 별도 외화 계좌나 환전 절차 없이 원화로 정산받는다. 해외 결제 수수료·환율 차이가 정산서에 명확히 표시되어 실수익을 바로 확인할 수 있다.

    카테고리 4. 확장 / 이전¶

    S4-01. 한국→일본 상품 자동 번역 + 번역 메모리¶

    일본 시장 진출을 결정한 셀러가 글로벌 진출 설정에서 '일본어' 언어를 추가한다. AI가 상품명·설명·카테고리명을 자동 번역하고 원문·번역문을 나란히 보여주는 검토 화면을 제공한다. 신뢰도 점수가 낮은 항목은 별도로 표시된다. 한 번 수정한 번역은 번역 메모리에 저장되어 이후 같은 표현이 나오면 자동으로 적용되므로 반복 수정이 줄어든다.

    S4-02. 한일 사이즈 자동 변환 — 구매자는 자기 나라 사이즈로 본다¶

    한국 사이즈(44/55/66/77)로 등록된 의류 상품에 일본 구매자가 접속한다. 국가 감지는 다중 신호 우선순위로 작동한다: ① 사용자 명시적 선택(쿠키/로컬스토리지 저장) → ② 기기 언어 설정(navigator.language) → ③ Accept-Language 헤더 → ④ IP 기반(fallback). 감지된 국가가 일본이면 상품 화면에서 사이즈가 자동으로 S/M/L/XL로 변환되어 표시된다. 상품 페이지에는 한일 사이즈 비교 팝업이 자동으로 추가되며, 팝업 내 '내 사이즈 기준 선택' 옵션을 통해 구매자가 원하는 사이즈 체계를 직접 선택하면 해당 값이 저장된다. 셀러는 상품을 따로 두 버전으로 등록할 필요 없이 한 번만 등록하면 된다.

    S4-03. 셀익스 연동 — 해외 주문이 오면 자동으로 처리¶

    글로벌 진출 설정에서 셀익스 API 키를 입력한다. 이후 해외 구매자의 주문이 들어오면 셀러가 별도로 처리하지 않아도 셀메이트CM이 셀익스로 주문 데이터를 자동 전달하고, 수출신고·라벨 발행·배송 상태가 실시간으로 주문 화면에 동기화된다. 상품명 + 소재 혼용률 + 편직 여부(Knitted/Woven) + 성별/연령 구분 + 카테고리 등 상품 속성을 종합해 AI가 HS코드를 자동 추천해 통관 오류 가능성도 줄여준다. 소재 혼용률 정보가 없는 경우 AI가 해당 정보를 입력 요청하거나 불확실성 경고를 표시한다.

    S4-04. 스토어 오픈 인스타 게시물 자동 생성¶

    스토어를 오픈한 셀러가 "인스타그램에 공유하기" 버튼을 누른다. 스토어 대표 이미지, 상품 사진, 웰컴 쿠폰 링크가 포함된 SNS 게시물 초안이 자동으로 생성되어 인스타그램 앱으로 바로 공유할 수 있는 상태가 된다. 셀러의 기존 팔로워가 오픈 소식을 확인하고 쿠폰 링크로 바로 구매할 수 있다.

    S4-05. 어필리에이트 추천 링크 — 지인이 구매하면 자동 보상¶

    셀러가 "추천인 링크 만들기"를 클릭하면 고유 추천 URL이 생성된다. 해당 링크를 통해 지인이 구매를 완료하면 구매자에게는 할인 쿠폰이 자동으로 지급된다. 셀러에게는 현금 리워드(정산금 추가 지급) 또는 플랜 크레딧(월정액 차감) 중 선택한 방식으로 보상이 지급된다. 또한 셀러가 다른 셀러를 셀메이트CM에 초대하는 리퍼럴 프로그램도 제공된다. 초대받은 셀러가 가입·첫 결제를 완료하면 소개 보상이 자동으로 지급된다. 셀메이트CM 어필리에이트 트래킹이 채널별 전환을 자동으로 집계하므로 셀러는 어떤 경로로 주문이 발생했는지 한눈에 파악할 수 있다.

    S4-06. 메이크샵·고도몰 마이그레이션 — CSV로 이전¶

    셀러가 메이크샵 등 기존 솔루션에서 내보낸 CSV 파일을 셀메이트CM에 업로드한다. 셀메이트CM이 CSV 컬럼을 AI로 분석해 셀메이트CM 필드에 자동 매핑하고, 매핑 결과를 확인 화면에 표시한다. 셀러는 AI 매핑 결과를 검토하고 필요 시 수정만 하면 된다. 업로드 실행 전 AI가 필수 필드 누락·가격 포맷 오류·이미지 URL 깨짐 등 사전 오류를 감지해 셀러에게 알려준다. 셀러가 오류를 확인하고 계속 진행 또는 수정을 선택한다. 문제없는 항목부터 순차 처리되고 오류 항목만 별도 CSV로 제공된다.

    카테고리 5. 문제 해결¶

    S5-01. 설정 변경 실수 → 24시간 내 롤백¶

    셀러가 AI 채팅으로 배송비 정책을 바꾼 직후 실수였음을 알아챈다. 모든 AI 채팅 설정 변경은 실행 전 자동으로 스냅샷이 저장되어 있어, "되돌리기" 버튼 하나로 24시간 이내에 이전 상태로 복구된다. 변경 이력은 로그로 남아 있어 언제 무엇이 바뀌었는지 추적할 수 있다.

    S5-02. 마이그레이션 정합성 검증 — 실제로 다 옮겼나 확인¶

    카페24에서 데이터 이전을 마친 셀러가 "정말 다 왔나?" 불안하다. 셀메이트CM이 이전 완료 후 자동으로 상품 수·이미지 접근 가능 여부·가격 오차·옵션 조합 수를 소스 데이터와 대조해 검증하고, 결과를 "1,234/1,247 완료, 오류 13건 (이미지 URL 불가)" 형태로 보여준다. 오류 목록은 CSV로 다운로드해 해당 항목만 수동으로 보완하면 된다.

    S5-03. 병행 운영 체크리스트 — 기존 몰 중단 없이 전환¶

    카페24 운영을 갑자기 멈출 수 없는 셀러가 셀메이트CM를 준비한다. 병행 운영 가이드에 따라 카페24는 주문을 계속 받으면서 셀메이트CM은 서브도메인(test.도메인.com)으로 테스트 환경을 운영한다. Go/No-Go 체크리스트(상품 등록 완료·테스트 결제 성공·배송비 시뮬레이션·DNS 준비)를 모두 통과한 뒤 DNS를 전환한다. 전환 권장 시간대(화~목 새벽 2~4시)도 안내된다.

    S5-04. 법령 변경 → 1-클릭 프리셋 업데이트¶

    전자상거래법 개정으로 소비자 고지 항목이 바뀐다는 소식이 들린다. 셀메이트CM이 법령 변경을 감지해 "한국 프리셋 업데이트가 있어요. 지금 적용하시겠어요?"라는 알림을 보낸다. 셀러가 확인 버튼을 누르면 이용약관·고지문·청약철회 정책이 최신 기준으로 자동 갱신된다. 사용자는 검토 후 게시 버튼을 누른다.

    S5-05. 테스트 주문 시뮬레이션 — 오픈 전 설정 오류 발견¶

    스토어를 오픈하기 전 셀러가 "테스트 구매" 버튼을 누른다. 실제 결제 없이 가상 주문이 생성되어 장바구니·결제 화면·주문 완료·발송 처리까지 전체 플로우를 직접 체험할 수 있다. 배송비 정책이 잘못 설정된 것을 발견하고 실구매자가 불편을 겪기 전에 수정한다. 테스트 주문은 완료 후 자동 삭제된다.

    S5-06. 72시간 미매출 → 1:1 온보딩 지원 연결¶

    스토어를 오픈했지만 72시간 동안 주문이 없는 셀러에게 셀메이트CM이 메시지를 보낸다. "아직 첫 주문을 기다리고 있네요. 전문 온보딩 코치가 1:1로 도와드릴 수 있어요." 셀러가 수락하면 CS 팀이 스토어를 직접 점검해 상품 구성·가격·결제 설정 중 전환율을 낮추는 요인을 찾아 피드백을 준다.

    카테고리 6. 첫 사용 여정¶

    S6-01. 처음 가입 → 30분 스토어 개설 전체 여정¶

    신규 셀러가 카카오로 가입하고 판매 카테고리(의류), 운영 국가(한국)를 선택하면 AI Setup Wizard가 자동으로 시작된다. 5단계(기본정보 → 테마 → 상품 → 결제 → 오픈 준비)를 순서대로 안내하며 각 단계마다 미리보기 패널에서 실제 스토어 화면을 확인할 수 있다. 이탈해도 진행 상태가 저장되어 재접속 시 멈춘 단계부터 이어서 진행되고, 마지막 단계 완료 시 스토어 URL이 생성되어 즉시 주문 수령 가능 상태가 된다.

    S6-02. 첫 상품 등록 → AI 초안 검토 → 게시 완료¶

    셀러가 스마트폰으로 찍은 상품 사진을 드래그해서 올리면 AI가 이미지를 분석해 상품명·상세설명·카테고리·추천 태그 초안을 2분 안에 자동 생성한다. 셀러는 AI가 생성한 초안을 검토하고 필요한 부분만 수정한 뒤 가격과 재고 수량을 입력한다. "게시" 버튼을 누르면 상품이 즉시 스토어에 노출되고, 검토 완료 알림이 어드민 홈 피드에 표시된다.

    S6-03. 결제 설정 완료 → 한국 PG 연동 (무통장입금 즉시 + 카드 PG 심사 대기 중)¶

    셀러가 Setup Wizard 결제 단계에서 "결제 수단 설정"을 진행하면 무통장입금·가상계좌가 즉시 활성화되어 PG 심사를 기다리지 않고 바로 주문을 받을 수 있다. 카드 PG 신청서를 작성해 제출하면 심사 진행 상태가 어드민 상단 배너에 단계별로 표시된다(서류 접수 → 심사 중 → 승인 완료). 심사 완료 시 카카오 알림톡이 발송되고, 알림 확인 후 클릭 한 번으로 카드 결제가 활성화된다.

    S6-04. 첫 주문 수령 → 발송 처리 → 구매자 배송 확인까지¶

    구매자가 결제를 완료하면 셀러에게 카카오 알림톡으로 주문 알림이 즉시 전송된다. 셀러는 어드민 주문 관리 화면에서 운송장 번호를 입력하고 발송 처리를 완료하면 구매자에게 배송 시작 알림톡이 자동 발송된다. 택배사 추적이 자동으로 연동되어 배송 완료 시 구매자에게 완료 알림이 전송되고, 셀러는 주문 상태가 "배송 완료"로 자동 전환된 것을 어드민에서 확인할 수 있다.

    카테고리 7. 해외몰 확장¶

    S7-01. 글로벌 대시보드에서 "일본 스토어 추가" 클릭 → 국가 설정 완료¶

    한국 스토어를 운영 중인 셀러가 글로벌 대시보드의 "스토어 추가" 버튼을 누르고 국가를 일본으로 선택한다. 일본 스토어 기본 설정(통화 엔화, 언어 일본어, 시간대 JST)이 자동으로 적용되며, 셀러는 스토어 이름과 운영 이메일만 입력하면 된다. 설정 완료 즉시 글로벌 대시보드에 일본 스토어 카드가 추가되고, 한국 스토어와 일본 스토어를 하나의 화면에서 전환하며 관리할 수 있는 상태가 된다.

    S7-02. 일본 결제 패키지 설정 (콘비니·코드결제·카드 자동 활성화)¶

    일본 스토어 설정 화면에서 "일본 결제 패키지" 토글을 켜면 GMO PG 기반으로 신용카드(JCB·VISA·Mastercard)·PayPay·콘비니 결제·코드결제가 한 번에 활성화된다. 별도 PG사 계약 없이 셀메이트CM를 통해 한국 법인으로도 일본 로컬 결제수단에 접근할 수 있으며, 이중 수수료 없이 GMO PG 단일 수수료만 부담한다. 활성화 후 테스트 결제 시뮬레이션으로 각 결제수단이 정상 작동하는지 확인할 수 있다.

    S7-03. 일본 배송 연동 (야마토 등 일본 택배사 프로필 설정)¶

    일본 스토어 배송 설정에서 야마토운수, 사가와택배 등 일본 주요 택배사 목록이 제공되고 셀러는 이용할 택배사를 선택한다. 택배사 계약 번호와 API 키를 입력하면 연동이 완료되어 일본 주문 발송 시 자동으로 해당 택배사의 운송장 번호가 발급된다. 배송 상태는 일본 구매자의 주문 확인 화면에 자동으로 표시되고, 배송 완료 시 구매자에게 LINE 또는 이메일로 완료 알림이 발송된다.

    S7-04. 한국 상품을 일본 스토어에 동기화 → 엔화 가격·일본어 설명 자동 설정¶

    셀러가 한국 스토어 상품 목록에서 일본 스토어에 노출할 상품을 선택하고 "일본 스토어 동기화" 버튼을 누른다. 선택한 상품의 설명이 AI로 자동 번역되어 일본어 초안이 생성되고, 원화 가격은 설정한 환율 정책(실시간 환율 + 마진율)에 따라 엔화로 자동 변환된다. 동기화된 상품은 일본 스토어에 즉시 노출되며, 이후 한국 스토어에서 상품 정보를 수정하면 동기화 대상 항목에 대해 일본 스토어에도 변경 적용 여부를 선택할 수 있다.

    카테고리 8. 번역 관리¶

    S8-01. 한국어 상품 1개를 일본어·영어로 AI 자동 번역 요청 → 초안 확인¶

    셀러가 상품 상세 편집 화면에서 "다국어 번역" 탭을 열고 번역 대상 언어(일본어, 영어)를 선택한 뒤 "AI 번역" 버튼을 누른다. AI가 상품명·상세설명·옵션명을 언어별로 번역해 원문과 번역문을 나란히 배치한 검토 화면을 약 30초 안에 표시한다. 번역 신뢰도가 낮은 표현(브랜드 고유 용어, 소재명 등)은 노란색으로 강조 표시되어 셀러가 우선적으로 검토하고 수정할 수 있다.

    S8-02. AI 번역 초안 검토 · 수정 → 번역 메모리에 저장 (동일 단어 일관성 유지)¶

    셀러가 번역 검토 화면에서 강조된 표현을 클릭하면 대체 번역 후보 목록이 드롭다운으로 표시된다. 셀러가 원하는 번역을 선택하거나 직접 입력하고 저장하면 해당 표현이 번역 메모리에 등록된다. 이후 동일한 원문 표현이 다른 상품에서 나타나면 번역 메모리의 값이 자동으로 적용되어 전체 스토어의 번역 일관성이 유지된다.

    S8-03. 전체 상품 일괄 번역 (100개) → 백그라운드 처리 + 완료 알림¶

    셀러가 상품 관리 목록에서 번역이 필요한 상품 100개를 체크하고 "일괄 번역" 버튼을 누른다. 번역 작업이 백그라운드로 시작되어 셀러는 브라우저를 닫거나 다른 작업을 해도 무방하며, 진행 상황은 어드민 상단 진행 바에서 실시간으로 확인할 수 있다. 번역이 모두 완료되면 카카오 알림톡으로 "100개 상품 번역 완료" 알림이 전송되고, 검토가 필요한 저신뢰도 항목 수와 바로가기 링크가 함께 제공된다.

    카테고리 9. AI 채팅 설정¶

    S9-01. "배송비 무료 기준을 5만원으로 바꿔줘" → AI가 확인 후 즉시 처리¶

    셀러가 어드민 AI 채팅(Ctrl+K)에 "배송비 무료 기준을 5만원으로 바꿔줘"라고 입력한다. AI가 현재 설정(항상 3,000원 고정)과 변경 후 내용(5만원 이상 무료)을 나란히 보여주며 "이대로 변경할까요?"라고 확인을 요청한다. 셀러가 확인을 누르면 즉시 적용되고, 24시간 이내에는 채팅 메시지 옆의 "되돌리기" 버튼으로 이전 설정으로 롤백할 수 있다.

    S9-02. "일본 스토어 운영 시간을 JST 기준 9~22시로 설정해줘" → AI가 처리¶

    셀러가 AI 채팅에 "일본 스토어 운영 시간을 JST 9시부터 22시로 설정해줘"라고 입력한다. AI가 현재 일본 스토어의 운영 시간 설정(24시간)을 보여주고 변경 내용을 요약해 확인을 요청한다. 승인 후 일본 스토어는 JST 기준 22:01부터 익일 08:59까지 "운영 시간 외" 안내 페이지를 표시하며, 해당 시간에 인입된 주문 문의는 다음 운영 시간에 자동 응답 메시지가 발송된다.

    S9-03. "지금 등록된 할인 쿠폰 전체 조회해줘" → AI가 목록 표시 후 특정 쿠폰 수정까지¶

    셀러가 AI 채팅에 "지금 등록된 할인 쿠폰 전체 보여줘"라고 입력하면 AI가 현재 활성·비활성 쿠폰 목록을 표 형태로 채팅창 안에 표시한다. 셀러가 목록에서 특정 쿠폰을 지목해 "이 쿠폰 할인율 10%에서 15%로 바꿔줘"라고 추가 요청하면 AI가 변경 내용을 확인 후 즉시 적용한다. 쿠폰 수정 이력은 변경 로그에 기록되며, AI 채팅 대화 기록에서 해당 변경 시점과 내용을 언제든지 다시 확인할 수 있다.

    카테고리 10. AI 채팅 분석¶

    S10-01. "이번 달 매출 왜 떨어졌어?" → AI가 원인 분석 (상품·채널·시간대별)¶

    셀러가 AI 채팅에 "이번 달 매출이 왜 떨어졌는지 분석해줘"라고 입력한다. AI가 지난달 대비 매출 변화를 상품별·채널별·시간대별로 분석해 주요 원인 3가지를 우선순위 순서로 정리한 인사이트 카드를 채팅창에 표시한다. 예: "1. 블라우스 카테고리 전환율 31% 하락 (재고 소진 후 품절 기간 9일) 2. 인스타그램 채널 트래픽 18% 감소 3. 주말 오후 3~6시 결제 완료율 저하" — 각 항목마다 "자세히 보기" 링크로 해당 리포트 화면으로 바로 이동할 수 있다.

    S10-02. "지난주 가장 잘 팔린 상품 5개 알려줘" → AI가 표 형태로 답변¶

    셀러가 AI 채팅에 "지난주 판매 1위부터 5위 상품 알려줘"라고 입력한다. AI가 지난 7일간 판매 데이터를 집계해 상품명·판매 수량·매출액·전주 대비 증감률을 포함한 표를 채팅창에 즉시 표시한다. 표 아래에는 "이 상품들의 재고를 미리 확보하시겠어요?" 등 다음 액션을 제안하는 버튼이 제공되어 분석에서 운영 실행으로 바로 연결된다.

    S10-03. "일본 스토어 결제 완료율이 낮은 이유 분석해줘" → AI가 결제 퍼널 분석¶

    셀러가 AI 채팅에 "일본 스토어 결제 완료율이 낮은 것 같아, 이유 분석해줘"라고 입력한다. AI가 지난 30일간의 결제 퍼널 데이터(상품 조회 → 장바구니 → 결제 시작 → 결제 완료)를 단계별 이탈률과 함께 표시하고, 이탈이 가장 높은 구간(예: 결제 수단 선택 → 결제 완료)의 원인을 추정한다. "PayPay 선택 후 이탈 41% — 결제창 로딩 3.2초 이상에서 집중 발생, 결제 UX 최적화 적용 시 완료율 약 8~12% 개선 예상"과 같은 구체적인 개선 방향과 예상 효과가 함께 제시된다.
