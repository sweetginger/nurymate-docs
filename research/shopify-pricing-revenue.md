# Shopify 요금제 및 수익 창출 구조 전수 조사 [4-13]

> 조사일: 2026-03-10 | 목적: 누리메이트 수익 모델 설계 벤치마크

---

## 1. 요금제별 월정액 및 포함 기능 (2025/2026 최신 기준)

### 1-1. 플랜 라인업 개요

| 플랜 | 월정액 (월별 결제) | 월정액 (연간 결제) | 타겟 |
|------|-------------------|-------------------|------|
| Starter | $5/월 | $5/월 (할인 없음) | 소셜 채널·링크 기반 판매 |
| Basic | $39/월 | ~$29/월 | 초기 창업자 |
| Shopify (Grow) | $105/월 | ~$79/월 | 성장 단계 쇼핑몰 |
| Advanced | $399/월 | ~$299/월 | 스케일업 단계 |
| Plus | $2,300/월~ | 3년 약정 시 협의 | 중견·엔터프라이즈 |
| Retail (POS 전용) | $89/월 | - | 오프라인 리테일 |

> 연간 결제 시 약 25% 할인 적용. 신규 가입자 대상 3일 무료 체험 + 3개월 €1/월 프로모션(일부 지역).

### 1-2. 플랜별 주요 포함 기능

#### Starter ($5/월)
- 소셜 미디어·채팅·링크 기반 판매
- 결제 처리 기능 포함
- 독립형 온라인 스토어 없음
- 주로 Instagram, Facebook, WhatsApp 등 채널 판매용

#### Basic ($39/월 / 연간 ~$29)
- 온라인 스토어 (무제한 상품 등록)
- 스태프 계정: 2명
- 기본 분석·보고서 (커스텀 보고서 불가)
- 블로그, 쿠폰, 선물 카드
- Shopify Shipping 이용 가능
- 판매 채널 연동 (Facebook, Instagram, Google 등)

#### Shopify/Grow ($105/월 / 연간 ~$79)
- 스태프 계정: 5명
- 표준 보고서 전체 열람 (커스텀 보고서 불가)
- 재고 위치: 5개소
- 배송 할인율 Basic 대비 향상

#### Advanced ($399/월 / 연간 ~$299)
- 스태프 계정: 15명
- **커스텀 보고서 빌더 + ShopifyQL 기본 사용**
- 고급 분석 필터
- 재고 위치: 8개소
- 제3자 결제 수수료 최저 (0.6%)
- 국제 가격 책정 (통화별 별도 가격 설정)

#### Plus ($2,300/월~ / 3년 약정)
- 스태프 계정: 무제한
- 재고 위치: 200개소
- **ShopifyQL Notebooks (전용 기능)**
- Shopify Flow (자동화 워크플로우 고급 기능)
- B2B 도매 채널 내장
- 맞춤형 결제 페이지 (Checkout Extensibility)
- 전용 기술 지원 매니저
- 최대 10개 스토어 관리 (멀티스토어)
- API 호출 한도 대폭 상향

## 2. 결제 수수료 구조

### 2-1. Shopify Payments 사용 시 (카드 처리 수수료)

| 플랜 | 온라인 결제 | 오프라인 결제 | 추가 Transaction Fee |
|------|------------|-------------|----------------------|
| Basic | 2.9% + $0.30 | 2.6% + $0.10 | 없음 |
| Shopify/Grow | 2.6% + $0.30 | 2.5% + $0.10 | 없음 |
| Advanced | 2.4% + $0.30 | 2.4% + $0.10 | 없음 |
| Plus | 2.15% + $0.30 (협의) | 협의 | 없음 |

> Shopify Payments는 미국, 캐나다, 영국, 호주, 일본, 독일 등 주요 국가에서만 제공.

### 2-2. 외부 PG(Third-Party Payment Provider) 사용 시 추가 Transaction Fee

| 플랜 | 추가 수수료 (Transaction Fee) |
|------|------------------------------|
| Starter | 5% |
| Basic | 2% |
| Shopify/Grow | 1% |
| Advanced | 0.6% |
| Plus | 0.2% |

- 외부 PG(Stripe, PayPal 등)를 사용할 경우 PG 수수료와 별도로 Shopify에 위 수수료를 추가 납부
- **일본·한국 등 Shopify Payments 미지원 국가에서는 모든 결제가 외부 PG 처리** → 수수료 부담 상승
- Shopify Plus 가입 시 Transaction Fee를 커스텀 협의로 낮출 수 있음

## 3. 추가 수익원 (Beyond Subscription)

### 3-1. 앱 스토어 수수료 (Revenue Share)

- 2025년 1월 기준 정책 변경: 개발자 평생 첫 $100만 매출까지 수수료 면제(연간 리셋 폐지)
- $100만 초과분부터 **15% 수수료** 징수 (과거 30% → 20% → 15%로 인하)
- 연간 앱 스토어 매출 $2,000만 이상 대형 파트너는 별도 협의
- 앱 스토어 내 8,000개 이상 앱 운영 → 수익 쉐어 규모 막대

### 3-2. 테마 스토어 수수료

- 테마 판매가: 유료 테마 $150~$400 수준
- 테마 개발자에게 **수익 쉐어** 구조 (공식 비율 비공개, 일반적으로 70~85% 개발자 수령)

### 3-3. Shopify Capital (금융 서비스)

- 판매 데이터 기반 **현금 선급 (Merchant Cash Advance)** 서비스
- 미국, 캐나다, 영국, 호주 등 일부 국가에서 제공
- 고정 이자 없음: 미래 매출의 일정 비율로 상환
- 예) $10,000 선급 → 상환 총액 $11,000 (Factor Rate 1.1x)
- Shopify 입장에서는 이자 수익 없이 수수료(Factor)로 수익 창출

### 3-4. Shopify Shipping

- UPS, USPS, DHL, Canada Post 등과 협약 → 할인 배송료 제공 후 차익 수취
- 기본 플랜부터 이용 가능, 플랜 상위일수록 할인율 증가
- Shopify가 배송 물량을 집계하여 운반사와 협상 → 마진 구조

### 3-5. Shopify Markets (글로벌 확장 도구)

- 복수 국가·통화·언어 쇼핑몰 운영 도구
- 국가별 세금·관세 자동 계산, 현지 결제 방식 지원
- Advanced 이상 플랜에서 완전 기능 제공
- 일부 기능(Markets Pro)은 추가 수수료 구조 (국제 거래 중개 마진)

### 3-6. POS 하드웨어

- Shopify POS Go: $399
- Shopify POS Terminal: $349~$499
- 카드 리더기(Tap, Chip & Swipe): $49
- 하드웨어 판매 자체 수익 + 오프라인 Payments 수수료
- 2025년 기준 오프라인 커머스 매출 $748M (YoY +27%)

### 3-7. Shopify Balance (금융 계좌)

- 판매 수익을 바로 이체받는 전용 계좌
- 캐시백(0.5~2%) 및 디빗 카드 제공 → 예치금 기반 수익

### 3-8. Shopify Audiences (광고 타겟팅)

- Plus 전용
- Shopify 네트워크 전체 고객 행동 데이터를 활용한 광고 타겟 공유
- Meta, Google Ads 등에 최적화된 타겟 리스트 제공 → 구독 가치 제고

## 5. 연간 결제 할인 및 Plus 가격 구조

### 5-1. 연간 결제 할인
- **일괄 25% 할인** 적용
  - Basic: $39 → $29/월 (연 $348)
  - Grow: $105 → $79/월 (연 $948)
  - Advanced: $399 → $299/월 (연 $3,588)

### 5-2. Plus 엔터프라이즈 가격
- 최소 **$2,300/월** (3년 약정 기준)
- 실제 계약 시 GMV 규모·기능 요구 사항에 따라 맞춤 협의
- 월 GMV $800K 이하: 정액 $2,300, 초과 시 GMV의 0.25% 부과 방식도 존재
- 대형 엔터프라이즈: Enterprise 커스텀 계약 (NDA 기반)

## 6. 누리메이트 수익 구조 설계 시 시사점

### 6-1. 구독 + 거래 수수료 병행 모델의 타당성
- Shopify는 구독 요금(정액) + 결제 수수료(변동) + 앱 생태계(수익 쉐어)의 3중 수익 구조
- 누리메이트도 월정액 구독만으로는 수익 한계 → **GMV 기반 수수료 또는 결제 처리 마진**을 추가 레이어로 설계 고려

### 6-2. 플랜 차등화 전략
- Basic → Advanced 상향 유도의 핵심 동인: **보고서/분석 기능 잠금**
- 누리메이트 역시 고급 통계·ShopifyQL류 쿼리 기능을 상위 플랜에 배치 → 자연스러운 업그레이드 유도 가능

### 6-3. 국내 시장 특수성 반영
- Shopify Payments 미지원 국가(한국 포함)에서 외부 PG 수수료 부담이 추가됨 → 누리메이트는 국내 PG사 직접 통합으로 Transaction Fee 면제 또는 최소화를 셀링포인트로 활용 가능

### 6-4. 금융 서비스(Capital)를 중기 확장 옵션으로
- Shopify Capital은 판매 데이터 기반 선급금으로 충성도 상승 효과
- 누리메이트가 충분한 판매 데이터를 축적한 이후, 국내 핀테크사와 제휴하여 유사 서비스 제공 가능

### 6-5. 앱 생태계 수익 쉐어
- Shopify 앱 스토어 수수료(15%)는 장기적으로 큰 수익원
- 누리메이트 앱 마켓플레이스 구성 시 수익 쉐어율을 초기에는 0~10%로 낮게 설정하여 파트너 생태계 구축 후 점진적 인상 전략 유효
