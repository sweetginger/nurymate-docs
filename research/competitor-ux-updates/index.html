    경쟁사 UX 업데이트 벤치마킹¶

    조사 기준일: 2026년 3월 관련 태스크: [15-1]

    조사 결과¶

    카페24¶

    최근 주요 업데이트 (2022~2026)¶

    • 2022년: 에디봇 AI 이미지 크롭 출시 — 쇼핑몰 최근 이미지 비율을 자동 조회·크롭, 모델 얼굴 인식 후 최적 편집. AI 이미지 자동 분류 추가 — 업로드 이미지를 모델컷/색상컷/디테일컷으로 자동 분류·배치. 에디봇 전시관 공개(70개+ 템플릿), 누적 상세페이지 제작 500만 회 돌파
    • 2023년: 리소스 잠금 아이콘 추가(삭제 방지). 에디봇 소셜(SNS 통합 콘텐츠 관리) 출시
    • 2024년 2월: AI 파트너사 42개사 제휴 — 이커머스 AI 기능 생태계 확장. 와들 'Gentoo(젠투)' 구매 전환 AI 에이전트 연동 포함
    • 2024년: 고객 정보 CSV 일괄 불러오기 추가. 카페24 허브 WMS 연동(셀메이트 없이도 연동). AI 앱 스토어 거래액 679억 원(AI 활용 앱 비중 약 20%)
    • 2025년 3월: 주문서 추가항목 API 신규 추가. HS코드 수정·조회. 결제수단 표기방식(텍스트/로고 아이콘) 선택 기능 개선
    • 2025년 하반기: AEO/GEO 시대 대응 SEO 고도화 — 언어별 메타데이터 자동 최적화, hreflang 자동 관리. 사이트 구조·브랜드 스토리·상품 카탈로그를 AI 친화 형식으로 자동 생성
    • 2026년 2월: 국내 이커머스 플랫폼 최초 오픈AI '앱스 인 챗GPT' 전용 앱 출시 — 챗GPT 내에서 카페24 스토어 상품 추천·비교·구매 가능

    에디봇 AI 기능 전체 목록 (2026년 기준)¶

    기능 설명
    AI 상세페이지 자동 생성 이미지 업로드 → AI가 카테고리 인식 → 템플릿 자동 적용 → 상세페이지 초안 생성. 100분→3분
    AI 이미지 자동 분류 업로드 이미지를 모델컷/색상컷/디테일컷으로 자동 분류·배치
    AI 이미지 크롭 쇼핑몰 비율 자동 감지 + 모델 얼굴 인식 기반 최적 썸네일 편집
    AI 상품 카테고리 인식 이미지에서 카테고리·색상 자동 인식
    에디봇 배너 쇼핑몰 배너 AI 자동 제작
    에디봇 소셜 SNS 통합 관리 + SNS 투고 콘텐츠 제작 지원
    에디봇 프로모션 이벤트 등록~통계 분석 일원화
    에디봇 리뷰 고객 상품 후기를 상세페이지에 자동 삽입
    에디봇 전시관 AI 기반 쇼핑몰 기획전·카테고리 전시 자동 구성

    ⚠️ 기존 문서에 기재된 "SNS 투고문·문의 대응문 자동 생성" 기능은 공식 자료에서 명시 미확인. 에디봇 소셜이 SNS 콘텐츠를 지원하나 LLM 기반 문장 생성 여부는 불명확.

    사용자 편의 개선 핵심 항목¶

    • 에디봇 AI로 상세페이지 제작 100분 → 3분 단축
    • AI 이미지 분류·크롭 자동화 (편집 공수 제거)
    • 챗GPT 앱 연동으로 AI 플랫폼 내 쇼핑 채널화 (AEO 대응)
    • 42개 AI 파트너 생태계로 기능 확장 가속

    누리메이트 대비 현황¶

    카페24는 에디봇(이미지·콘텐츠 AI)과 AI 생태계 파트너십에서 2022년부터 앞서 있으나, AI 스토어 설정 자동화·온보딩 마법사 미보유. 챗GPT 채널 연동은 누리메이트의 AEO/GEO 전략과 겹치는 영역으로 선점 속도 주시 필요. 일본 결제·크로스보더는 별도 앱 연동 구조로 누리메이트 네이티브 지원 대비 열위. 통합 정산 대시보드 미제공.

    아임웹¶

    최근 주요 업데이트 (2022~2025)¶

    • 2024년 4월: 주문관리시스템(OMS) 전면 리뉴얼 — 공식 명칭 '주문관리 v2'. 2024년 4월 24일부터 순차 적용, 6월 전체 완료. 하기 OMS 상세 항목 참조
    • 2024년 5월: CRM 캠페인 자동화 출시 — 회원가입 후 1일 미구매, 장바구니 1시간 방치, 1개월 재구매 없음 등 행동 트리거별 카카오톡 메시지·쿠폰 자동 발송. 출시 1개월 내 구매 전환율 평균 85% 향상 측정
    • 2024년 8월: '전문가 찾기' 서비스 출시 — 쇼핑몰 제작·운영 전문가와 셀러를 연결하는 마켓플레이스
    • 2025년: AI 사이트 빌더 '아임AI' 출시. AI 기술지원 챗봇 'BIBI(비비)' 출시. 누적 개설 사이트 100만 개·누적 거래액 6조~7조 원 돌파

    OMS 전면 리뉴얼 상세 (주문관리 v2, 2024년 4월)¶

    1. 주문 목록 통합 - 기존: 결제 상태·배송 상태별로 메뉴 분산 → 리뉴얼: 주문 메뉴 하나에서 전체 조회 가능 - 세분화된 검색 필터, 대량 주문 일괄 처리 지원

    2. 타임라인 이력 관리 - 주문별 전체 처리 이력(접수→결제→배송→완료)을 시간 순으로 시각화 - 담당자 메모 기능 포함

    3. 추가 결제 요청 (아임웹 독자 기능) - UX 흐름: 관리자가 품목·수량 수정 → 추가 금액 산출 → "고객 마이페이지로 추가 결제 요청" 클릭 → 고객 알림 발송 → 고객이 마이페이지에서 직접 결제 완료 - 기존 방식(가상계좌 안내, 별도 링크 생성)을 완전 대체. 카페24·고도몰은 이 흐름 미지원

    4. 부분 처리 강화 - 하나의 주문에서 일부 품목·수량만 취소·반품·교환 가능 - 교환 승인 시 동일 상품 자동 추가(상품 준비 상태)로 전환 → 기존 배송 흐름으로 진행 - 부분 취소·반품 시 변동 금액은 마이페이지 추가 결제 요청 기능 연동

    5. 취소·환불 흐름 표준화 - [취소 접수 → 취소 승인 → 환불] 3단계 표준화 - 자동 환불: 카드결제·카카오페이 등 PG 취소 승인 시 자동 처리 - 수동 환불: 무통장입금·가상계좌·휴대폰 결제는 관리자 수동 이체 후 처리

    아임AI — AI 사이트 자동 생성¶

    • UX 흐름: 콘셉트 텍스트 입력 → 사이트명·URL 입력 → AI 생성(약 3분 소요) → 미리보기 → 공개 → 요금제 선택(14일 무료 체험)
    • 생성 후 어드민에서 모든 디자인·콘텐츠 자유 편집 가능
    • 제약: 하루 1회 생성 제한. 생성 중 페이지 이탈 시 사이트 삭제. 프롬프트 품질에 따라 결과 차이 큼

    BIBI(비비) — AI 기술지원 챗봇 상세¶

    • 기술 구조: 채널톡 기반 ❌ — 아임웹 자체 개발. 1,300여 개 아임웹 공식 기술 문서 기반 RAG(검색 증강 생성) 방식으로 구현
    • 외부 범용 챗봇 도입 시 "정확히 매칭되는 질문만 답변 가능"한 한계를 극복하기 위해 자체 문서 탐색 방식 채택
    • 사용 LLM 모델(GPT-4, Claude 등)은 공식 공개 미확인
    • 접속: bb.imweb.me — 아임웹 계정 로그인 후 이용
    • 기능 범위: 아임웹 사용법 질문 답변 (어드민 기능, 디자인 모드, 주문 관리 방법 등). 자연어로 질문하면 즉시 답변
    • 중요 한계: BIBI는 아임웹 운영자(셀러) 대상 기술지원 챗봇임. 쇼핑몰 방문 고객 대상 CS 챗봇이 아님. 고객 응대용 챗봇은 채널톡 등 외부 앱을 별도 설치해야 함

    사용자 편의 개선 핵심 항목¶

    • OMS 마이페이지 추가 결제 요청으로 주문 수정 후속 결제 마찰 제거 (아임웹 독자 기능)
    • CRM 자동화로 구매 유도 메시지 자동 처리 (전환율 +85%)
    • 아임AI로 비개발자 대화형 쇼핑몰 생성 (~3분)
    • BIBI로 어드민 사용법 문의 AI 즉시 응대 (CS 공수 절감)

    누리메이트 대비 현황¶

    아임웹 OMS의 마이페이지 추가 결제 요청은 카페24·고도몰에 없는 독자 기능으로 누리메이트 주문 관리 기획 시 참고 필요. BIBI의 자체 RAG 방식은 누리메이트 M-09 AI 채팅의 기술적 방향과 유사. 아임웹은 한국 단일 시장 기반으로 일본 결제·다국어 지원 없음 — 이 점이 누리메이트의 핵심 차별점.

    • 2024년 12월: STORES 모바일 오더 출시 — 실점포에서 고객이 스마트폰으로 QR코드 스캔 후 주문·결제 완결하는 시스템. 초기 비용·월정액 0엔
    • 2025년 3월: 모바일 오더에 PayPay 결제 추가
    • 2025년 4월: 키친 디스플레이 앱 연동 + 단말 무상 제공 캠페인
    • 2025년 3월: 통합 올인원 플랜 출시 — POS레지·키캐시리스결제·네트샵·예약시스템·모바일오더·데이터분석 7개 서비스를 月3,300엔·결제수수료 1.98%~로 단일화
    • 2025년 8월: 모바일 오더 테이블 오더 기능 추가 (月7,700엔 별도)
    • 2025년 가을: 결제 단말기 2세대 출시. 레지 앱 iPhone 완전 대응. 영수증으로도 주문 관리 가능. 30개+ 항목 개선

    STORES 모바일 오더 상세¶

    기능 정의: 실점포에서 고객이 자신의 스마트폰으로 QR코드를 스캔해 주문부터 결제까지 완결하는 비대면 주문 시스템.

    기능 내용
    테이크아웃 주문 QR 스캔 → 메뉴 선택 → 결제 완료 (카운터 없이 처리)
    테이블 오더 테이블에 QR 설치 → 착석 상태에서 주문·결제 완결 (2025년 8월 추가)
    PayPay 결제 신용카드 외 PayPay 결제 지원 (2025년 3월 추가)
    키친 디스플레이 연동 주방 전용 앱에 주문 자동 수신
    STORES POS 자동 연동 레지스터와 데이터 동기화, 재고 자동 관리
    옵션 커스터마이징 토핑 등 옵션, 품절 원터치 정지, 시간대별 판매 스케줄
    스태프 대리 주문 스태프가 고객 대신 주문 입력 가능

    주 타겟 업종: 음식점, 카페, 커피숍, 테이크아웃 비중 높은 점포, 인력 부족 소규모 점포

    요금 구조:

    항목 내용
    초기 비용·월정액 0엔
    결제 수수료 3.6% + 10엔/건
    스탠다드 플랜(月3,300엔) 포함 모바일 오더 일부 기능 포함
    테이블 오더 月7,700엔(세금 포함) 별도 추가
    스탠다드 + 유료 레지 병용 시 月2,000엔 할인

    소비자 조사 (2025년 3월, STORES 자체): 모바일 오더 이용 가능 매장에서 약 70%의 이용자가 매장 방문 빈도 증가 응답.

    사용자 편의 개선 핵심 항목¶

    • 모바일 오더로 실점포 비대면 주문 완결 (인력 절감·고객 대기 단축)
    • 테이블 오더로 착석 후 주문까지 스마트폰 완결 (서빙 효율화)
    • 7개 서비스 단일 플랜 통합으로 개별 계약 복잡성 제거
    • 오프라인+온라인 매출 합산 분석 (O2O 운영 셀러 편의)
    • 원가 등록으로 수익률 관리 가능

    누리메이트 대비 현황¶

    STORES 모바일 오더는 오프라인 실점포 특화 기능으로 누리메이트 온라인 자사몰 포지션과 직접 경쟁하지 않음. STORES 통합 분석·O2O 단일 플랜 전략은 누리메이트 통합 정산 대시보드(P1)와 방향이 같으나 오프라인 연동은 누리메이트 로드맵 외 영역. STORES는 일본 시장 전용으로 한국 진출 없음.

    핵심 인사이트¶

    1. AI 기능이 경쟁의 중심축으로 완전히 이동했다. Shopify(Sidekick·Magic·AI 스토어 빌더), 아임웹(아임AI·BIBI), BASE(AIアシスタント), 카페24(에디봇 확장), 메이크샵(AI 상품 설명)까지 7개 플랫폼 전부가 2023~2025년 사이 AI 기능을 핵심 업데이트로 밀고 있다. 누리메이트가 AI 온보딩·AI 프론트빌더를 P0으로 배치한 방향은 맞으나, 출시 직후부터 이 기능이 실제로 작동해야만 "AI 플랫폼"이라는 포지셔닝이 유효하다.

    2. 소셜 커머스 연동이 급속히 표준 기능화되고 있다. BASE의 TikTok Shop API 연동(2025.7), YouTube 쇼핑 연동(2024.5), 카페24의 SNS 연동 앱 생태계 확장 등 판매 채널이 자사몰 단독에서 SNS 내 구매로 확장되는 속도가 빠르다. 누리메이트의 SNS 마케팅 연동(P1)을 런칭 직후 6개월 내 구현하지 않으면 경쟁력 격차가 벌어진다.

    3. 정산 속도와 어드민 분석이 셀러 락인의 핵심 변수로 부상했다. BASE의 최속 10분 정산, STORES의 통합 데이터 분석, 카페24의 대시보드 강화는 모두 "돈이 빠르게 들어오고, 데이터로 판단할 수 있는" 운영 편의를 강화하는 방향이다. 누리메이트의 통합 결제 대시보드(P1)와 즉시 정산 구조 설계는 경쟁 플랫폼 대비 늦지 않게 Phase 2 내 구현이 필요하다.
