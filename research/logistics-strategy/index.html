    배송·물류 전략 조사¶

    작성일: 2026-03-11 관련 태스크: [12-6] 상태: Draft

    1. 카페24 매일배송 서비스 조사¶

    1.1 서비스 개요¶

    카페24 매일배송은 오늘 구매 → 내일 배송(익일 배송)을 보장하는 서비스다. 카페24와 계약한 3PL 파트너사(두손컴퍼니 등)가 카페24 주문을 받아 당일 출고 처리한다.

    항목 내용
    서비스 유형 익일 배송 보장 (오늘 주문 → 내일 도착)
    운영 방식 카페24 + 3PL 파트너 협력 구조
    3PL 파트너 두손컴퍼니 등
    서비스 지역 수도권 우선 출발, 전국 순차 확대

    1.2 이용 조건¶

    • 카페24 유료 플랜(Pro 이상) 이용자
    • 계약 3PL 창고에 재고 사전 입고 필수
    • 주문 마감 시간(오후 2시 등) 이전 주문 건만 당일 출고

    1.3 수수료 및 비용 구조¶

    • 3PL 창고 보관료: 팔레트/CBM 기준 별도 과금
    • 출고 처리비: 건당 별도 (상품 크기·중량에 따라 상이)
    • 배송비: 택배사 계약 단가 (건당 2,500~3,500원 수준)

    1.4 강점·약점¶

    강점 약점
    "내일 도착" 배지로 전환율 향상 3PL 창고에 재고 선투자 필요
    판매자 물류 부담 제거 소규모 판매자 최소 물량 기준 진입 장벽
    카페24 어드민 통합 운영 수도권 이외 지역 배송 속도 저하

    2. 한국 주요 3PL사¶

    3PL사 특화 영역 셀메이트CM 파트너십 가능성
    두손컴퍼니 카페24 파트너, 당일출고 특화 Phase 2 파트너십 후보 1순위
    팀프레시 새벽배송, 식품·신선식품 강점 식품 버티컬 확장 시 검토
    위킵(Wekip) 스타트업 전용 소규모 재고 특화 소규모 신규 창업자 대상 적합
    에스제이그룹 중견기업 규모 물류 중견 브랜드 셀러 대상
    로지스팟 API 기반 물류 중개 플랫폼 다중 3PL 연동 구조 검토

    3. 일본 주요 3PL사¶

    3PL사 특화 영역 셀메이트CM 활용 가능성
    EC-RIDER (イーシーライダー) 일본 EC 풀필먼트 전문 일본 셀러 풀필먼트 1순위
    Fulfillment by Rakuten 라쿠텐 연동 강점 라쿠텐 이탈 셀러 자사몰 전환 시 연계
    ヤマト運輸 (야마토 운수) 일본 최대 택배, B2B 물류 API 제공 배송 API 연동 최우선
    佐川急便 (사가와) 사가와 급편, 물류 API 제공 야마토 대체 옵션
    日本郵便 (일본우편) 전국망 + 소형 패키지 강점 소형 패키지 옵션

    4. 셀메이트CM 배송 전략¶

    Phase 1 — 판매자 자체 발송 (2026 H1 런칭)¶

    판매자가 직접 택배사에 접수하는 구조. 셀메이트CM은 택배 API를 통한 운송장 자동 생성과 배송 추적 통합을 제공한다.

    한국 지원 택배사: - CJ대한통운 (국내 1위, API 제공) - 한진택배 (API 제공) - 롯데택배 (API 제공) - 우체국택배 (e-POST API 제공)

    운송장 자동 출력: - 판매자 어드민에서 운송장 번호 입력 → 자동 배송 추적 연동 - Phase 2에서 운송장 자동 생성 (택배사 API 직접 연동)

    Phase 2 — 3PL 파트너십 (2026 H2)¶

    두손컴퍼니 등 3PL 공식 파트너 연결 서비스 제공. 판매자가 셀메이트CM 어드민에서 3PL 풀필먼트 신청 가능.

    • 파트너 3PL사 목록 및 조건 비교 UI 제공
    • 3PL 재고 입고 상태 셀메이트CM 어드민 연동
    • 픽업-포장-출고 자동 연동 (3PL API)

    Phase 3 — 일본 배송 연동 (2026 H2~2027)¶

    • 야마토 운수 ビジネスメンバーズ API 연동
    • 사가와 Biz API 연동
    • 일본우편 ゆうパック API 연동
    • 일본 콘비니 픽업 수령 연동 (spec/japan-conveni-pickup.md 참조)
